- **효과**: 더 컴팩트하고 국제적인 UI, 높은 접근성

## 접근성 준수 사항

### WCAG 2.1 AA 기준 확인

- ✅ **색상 대비**: 모든 텍스트와 UI 요소에서 최소 4.5:1 이상
- ✅ **포커스 표시**: 모든 인터랙티브 요소에 명확한 포커스 아웃라인
- ✅ **키보드 네비게이션**: Tab 키로 모든 요소 접근 가능
- ✅ **라이트/다크 모드**: 양쪽 모드에서 동일한 접근성 유지
- ✅ **반응형 디자인**: 모바일/태블릿/데스크톱 모두 최적화
- ✅ **ARIA 속성**: 언어 선택자에 `aria-label` 추가
- ✅ **아이콘 텍스트**: 모든 아이콘에 대체 텍스트 제공

### 향상 계획 (WCAG 2.1 AAA)

- 추가 텍스트 그림자로 더욱 부드러운 가독성
- 더 큰 터치 타겟 (모바일): 최소 44x44px
- 동작 감소 모드 (`prefers-reduced-motion`) 지원 강화
- 고대비 모드 (`prefers-contrast`) 지원 최적화
