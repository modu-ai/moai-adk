# 문서 네비게이션 가이드

MoAI-ADK 문서는 14개의 주요 카테고리로 구성되어 있습니다. 이 가이드는 각 섹션의 내용과 효과적인 문서 탐색 방법을 안내합니다.

## 문서 구조 개요

### 📚 학습 경로별 추천 순서

#### 초보자 경로 (Beginner Path)

1. **소개** → **시작하기** → **핵심 기능** → **실전 튜토리얼** → **코드 예제**

#### 중급자 경로 (Intermediate Path)

1. **Alfred 슈퍼에이전트** → **에이전트 팀** → **Skills 시스템** → **심화 가이드** → **사례 연구**

#### 고급자 경로 (Advanced Path)

1. **아키텍처** → **API 레퍼런스** → **심화 가이드** → **기여 가이드**

---

## 카테고리별 상세 안내

### 1. 소개 (index)

**대상**: 모든 사용자
**목적**: MoAI-ADK의 핵심 개념과 철학 이해

**포함 내용**:
- MoAI-ADK란 무엇인가?
- SPEC-First TDD 방법론
- 프로젝트 비전과 목표
- 빠른 시작 가이드

**추천 순서**: 가장 먼저 읽어야 할 페이지

### 2. 시작하기 (getting-started)

**대상**: 신규 사용자
**목적**: 빠르게 프로젝트를 시작하고 첫 성공 경험 만들기

**포함 내용**:
- 설치 및 환경 설정
- 첫 프로젝트 생성
- 기본 명령어 사용법
- Hello World 예제

**학습 시간**: 약 30분

### 3. 핵심 기능 (features)

**대상**: 모든 사용자
**목적**: MoAI-ADK의 주요 기능 이해

**포함 내용**:
- SPEC-First 개발
- TDD (Test-Driven Development)
- 에이전트 기반 워크플로우
- 자동화된 문서 동기화
- GitFlow 통합

**학습 시간**: 약 1시간

### 4. Skills 시스템 (skills)

**대상**: 중급 이상 사용자
**목적**: Skills 시스템의 구조와 활용법 마스터

**포함 내용**:
- Skills 개요 및 아키텍처
- Foundation Skills (15개 핵심 Skills)
- BaaS 생태계 통합
- 고급 Skills 활용
- Custom Skill 개발
- 검증 시스템

**하위 섹션**:
- `overview`: Skills 시스템 전체 개요
- `ecosystem-upgrade-v4`: v4.0 업그레이드 가이드
- `baas-ecosystem`: BaaS 플랫폼 통합
- `advanced-skills`: 고급 활용 패턴
- `validation-system`: 품질 검증 시스템
- `skill-development`: Custom Skill 개발
- `foundation`: 15개 Foundation Skills 상세

**학습 시간**: 약 2-3시간

### 5. Alfred 슈퍼에이전트 (alfred)

**대상**: 모든 사용자
**목적**: Alfred의 역할과 사용법 이해

**포함 내용**:
- Alfred의 4단계 워크플로우
- 명령어 시스템 (`/alfred:0-project`, `/alfred:1-plan`, 등)
- Expert Delegation System
- 에이전트 오케스트레이션

**하위 섹션**:
- `workflow`: 4단계 워크플로우 상세
- `commands`: 명령어 레퍼런스
- `agents`: 에이전트 관리
- `expert-delegation-system`: 전문가 위임 시스템

**학습 시간**: 약 1.5시간

### 6. 에이전트 팀 (agents)

**대상**: 중급 이상 사용자
**목적**: 19명의 에이전트 팀원 이해 및 활용

**포함 내용**:
- 10개 Core Agents
- 6개 Specialist Agents
- 2개 Built-in Agents
- 에이전트 간 협업 패턴
- 에이전트 선택 가이드

**학습 시간**: 약 2시간

### 7. 실전 튜토리얼 (tutorials)

**대상**: 초보~중급 사용자
**목적**: 실제 프로젝트 시나리오를 통한 hands-on 학습

**포함 내용**:
- **Tutorial 1**: REST API 개발
- **Tutorial 2**: JWT 인증 구현
- **Tutorial 3**: 데이터베이스 최적화
- **Tutorial 4**: Supabase BaaS 통합
- **Tutorial 5**: MCP 서버 개발

**특징**: 단계별 코드 예제 포함, 실행 가능한 프로젝트

**학습 시간**: 각 튜토리얼당 1-2시간

### 8. 사례 연구 (case-studies)

**대상**: 중급 이상 사용자
**목적**: 실제 프로덕션 사례를 통한 베스트 프랙티스 학습

**포함 내용**:
- **E-commerce 플랫폼**: 대규모 전자상거래 시스템 개발
- **Enterprise SaaS 보안**: 엔터프라이즈급 보안 구현
- **Microservices 전환**: 모노리스에서 마이크로서비스로 전환

**특징**:
- 실제 프로덕션 환경 경험
- 문제 해결 과정 상세 설명
- 성능 메트릭 및 결과 포함

**학습 시간**: 각 사례당 2-3시간

### 9. 코드 예제 (examples)

**대상**: 모든 사용자
**목적**: 복사-붙여넣기 가능한 실전 코드 스니펫

**포함 내용**:
- REST API 패턴
- 데이터베이스 CRUD 작업
- 인증/인가 구현
- 테스팅 전략
- 성능 최적화
- 보안 베스트 프랙티스

**특징**: 즉시 사용 가능한 코드 스니펫, 주석 포함

**학습 시간**: 참조용 (필요시 검색)

### 10. 심화 가이드 (guides)

**대상**: 중급 이상 사용자
**목적**: 고급 주제 및 복잡한 시나리오 마스터

**포함 내용**:
- 고급 SPEC 작성 기법
- 복잡한 TDD 시나리오
- 멀티 에이전트 협업 패턴
- 성능 최적화 전략
- 대규모 프로젝트 관리

**학습 시간**: 각 가이드당 1-2시간

### 11. 아키텍처 (architecture)

**대상**: 고급 사용자, 아키텍트
**목적**: MoAI-ADK 내부 아키텍처 이해

**포함 내용**:
- 4-Layer 아키텍처 (Commands → Agents → Skills → Hooks)
- Agent Orchestration 메커니즘
- Progressive Disclosure 패턴
- Context Budget 최적화
- 확장성 및 플러그인 시스템

**학습 시간**: 약 3-4시간

### 12. API 레퍼런스 (reference)

**대상**: 모든 사용자 (참조용)
**목적**: 완전한 API 문서 제공

**포함 내용**:
- CLI 명령어 레퍼런스
- Python API 문서
- Skill API 명세
- Agent API 인터페이스
- 설정 파일 스키마

**특징**: 자동 생성된 API 문서, 검색 최적화

### 13. 문제 해결 (troubleshooting)

**대상**: 모든 사용자
**목적**: 일반적인 문제 및 해결 방법

**포함 내용**:
- 설치 문제
- 실행 오류
- 에이전트 관련 문제
- Git/GitFlow 이슈
- 성능 문제
- FAQ

**특징**: 검색 가능한 에러 메시지, 단계별 해결 방법

### 14. 기여 가이드 (contributing)

**대상**: 컨트리뷰터
**목적**: 오픈소스 기여 방법 안내

**포함 내용**:
- 개발 환경 설정
- GitFlow 워크플로우
- 코딩 스타일 가이드
- PR 체크리스트
- 커밋 메시지 규칙
- 코드 리뷰 프로세스

**학습 시간**: 약 1시간

### 15. 출력 스타일 (output-style)

**대상**: 모든 사용자
**목적**: Alfred 출력 형식 커스터마이징

**포함 내용**:
- 리포트 형식 설정
- 언어 설정
- 출력 템플릿 커스터마이징

## 효과적인 학습 전략

### 1단계: 기초 다지기 (1-2일)

1. **소개** 읽기 (30분)
2. **시작하기**로 환경 설정 (30분)
3. **핵심 기능** 훑어보기 (1시간)
4. **Tutorial 1** 따라하기 (1-2시간)

### 2단계: 핵심 개념 마스터 (3-5일)

1. **Alfred 슈퍼에이전트** 깊이 이해 (2시간)
2. **Skills 시스템** 학습 (3시간)
3. **Tutorial 2-3** 완료 (3-4시간)
4. **코드 예제** 탐색 및 실습 (2시간)

### 3단계: 고급 활용 (1-2주)

1. **에이전트 팀** 전체 이해 (2시간)
2. **사례 연구** 분석 (6시간)
3. **심화 가이드** 학습 (4-6시간)
4. **Tutorial 4-5** 완료 (3-4시간)

### 4단계: 마스터 레벨 (지속적)

1. **아키텍처** 상세 이해 (4시간)
2. **기여 가이드** 읽고 기여 시작
3. **API 레퍼런스** 참조하며 고급 기능 활용
4. 커뮤니티 참여 및 지식 공유
