# 시작하기

MoAI-ADK를 시작하는 방법을 단계별로 안내합니다.

## 빠른 시작

### 1. 설치

```bash
# uv 사용 (권장)
uv pip install moai-adk

# pip 사용
pip install moai-adk
```

### 2. 프로젝트 초기화

```bash
moai-adk init
```

### 3. 첫 번째 워크플로우 실행

```bash
# Claude Desktop에서
/alfred:0-project
```
