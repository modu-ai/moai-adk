## 🛠️ 핵심 도구

### Alfred 슈퍼에이전트
MoAI-ADK의 핵심으로, 19개 전문 AI 에이전트를 오케스트레이션합니다.

- **`/alfred:0-project`**: 프로젝트 설정 및 초기화
- **`/alfred:1-plan`**: SPEC 작성 및 계획 수립
- **`/alfred:2-run`**: TDD 기반 자동 구현
- **`/alfred:3-sync`**: 문서 자동 동기화

### TRUST 5 원칙
MoAI-ADK의 품질 보증 시스템:

1. **Test-first**: 항상 테스트부터 작성
2. **Readable**: 명확하고 이해하기 쉬운 코드
3. **Unified**: 일관된 코드 스타일
4. **Secured**: 보안 검증 및 취약점 방지
5. **Trackable**: 완전한 추적성 보장
