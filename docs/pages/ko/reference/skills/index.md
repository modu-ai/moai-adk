# 스킬

사용 가능한 스킬과 그것들을 효과적으로 사용하는 방법을 발견하세요.

## 개요

스킬은 표준화된 템플릿, 모범 사례, 체크리스트를 제공하는 재사용 가능한 지식 캡슐입니다.

## 기본 스킬

- 필수 요소: 디버깅, 리팩토링, 성능 최적화
- 언어 감지: 자동 언어 감지
- 모범 사례: 품질 원칙 및 가이드라인

## 에이전트 스킬

- 에이전트 가이드: 선택 및 협업 패턴
- 문서 관리: 생성 및 동기화
- 품질 보장: TRUST 원칙 집행
