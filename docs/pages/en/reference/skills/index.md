# Skills

Discover the available skills and how to use them effectively.

## Overview

Skills are reusable knowledge capsules that provide standardized templates, best practices, and checklists.

## Foundation Skills

- Essentials: Debugging, refactoring, and performance optimization
- Language Detection: Automatic language detection
- Best Practices: Quality principles and guidelines

## Agent Skills

- Agent Guide: Selection and collaboration patterns
- Documentation: Management and synchronization
- Quality Assurance: TRUST principles enforcement
