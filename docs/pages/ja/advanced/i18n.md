# 多言語サポート

MoAI-ADKは多言語ドキュメントシステムを通じて、世界中の開発者に最高の開発体験を提供します。

## サポート言語

- **韓国語**（デフォルト）- 完全サポート
- **英語** - 完全サポート
- **日本語** - 完全サポート
- **中国語** - 完全サポート

## 自動言語検出

ユーザーのブラウザ言語設定に基づいて、適切な言語バージョンを自動的に表示します。

## ドキュメント構造

```
docs/src/
├── index.md           # 韓国語（デフォルト）
├── en/               # 英語ドキュメント
│   └── index.md
├── ja/               # 日本語ドキュメント
│   └── index.md
└── zh/               # 中国語ドキュメント
    └── index.md
```

## 翻訳ガイドライン

1. **一貫性**: すべての技術用語に一貫した翻訳を使用
2. **文化的適応**: 各言語の文化的特性に合わせて表現を調整
3. **更新同期**: 新機能追加時にすべての言語を同時に更新

## 貢献

新しい言語の追加や既存の翻訳の改善に興味がある場合は、[貢献ガイド](contributing/index.md)を参照してください。
