# MoAI-ADK 한국어 문서

**MoAI-ADK (Agentic Development Kit)** 는 Claude Code 환경에서 Alfred SuperAgent와 12개 이상의 서브 에이전트가 협업해 **SPEC → TDD → 코드 → 문서**를 자동으로 잇는 개발 프레임워크입니다. 이 온라인 문서는 한국어 기반으로 MoAI-ADK를 빠르게 이해하고 실무에 적용할 수 있도록 구성되어 있습니다.

## 왜 Alfred 기반 Agentic 워크플로우인가?

- **SPEC-First 철학**: 기능을 시작하기 전에 요구사항을 명확하게 구조화하여 실패를 줄입니다.
- **RED → GREEN → REFACTOR 자동화**: `/alfred:2-run`이 테스트 작성부터 구현, 리팩토링까지 주도합니다.
- **Living Documentation**: `/alfred:3-sync`로 README, CHANGELOG, 보고서, TAG가 항상 최신 상태를 유지합니다.
- **TRUST 5 원칙 준수**: Test, Readable, Unified, Secured, Trackable을 모든 단계에서 검증합니다.
- **TAG 체계 추적성**: @SPEC ↔ @TEST ↔ @CODE ↔ @DOC 체인이 연결되어 변경 영향도를 즉시 파악합니다.

## 지금 바로 시작하기

```bash
uv tool install moai-adk
moai-adk init hello-world
claude
/alfred:0-project
```

이후 `/alfred:1-plan "기능 설명"`으로 SPEC을 만들고, `/alfred:2-run SPEC-ID` → `/alfred:3-sync` 순서로 완전한 기능을 완성하세요.
