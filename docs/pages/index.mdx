# 🎩 MoAI-ADK

**SPEC-First TDD 개발 프레임워크 (Alfred 슈퍼에이전트 포함)**

MoAI-ADK는 명세 기반 테스트 주도 개발(TDD)을 체계적으로 지원하는 개발 프레임워크입니다. 🎩 Alfred 슈퍼에이전트가 개발 워크플로우를 자동화하여 높은 품질의 소프트웨어를 효율적으로 구축할 수 있도록 돕습니다.

## ✨ 주요 기능

### 🎯 SPEC-First 개발
- **SPEC 문서**: 명세서가 테스트와 코드를 주도합니다
- **@TAG 시스템**: 코드, 테스트, 문서 간 추적성 보장
- **자동화된 워크플로우**: SPEC → TDD → Sync

### 🤖 Alfred 슈퍼에이전트
- **19명의 팀 멤버**: 10명의 핵심 에이전트 + 6명의 전문가 + 2명의 빌트인 에이전트
- **55개 Claude Skills**: 특화된 도메인 전문성
- **4단계 워크플로우**: 의도 파악 → 계획 수립 → 작업 실행 → 보고 및 커밋

### 🔧 개발 도구 통합
- **CLI 명령어**: `/alfred:0-project`, `/alfred:1-plan`, `/alfred:2-run`, `/alfred:3-sync`
- **테스트 자동화**: RED → GREEN → REFACTOR
- **Git 워크플로우**: 자동 브랜치 관리 및 PR 생성

## 🚀 빠른 시작

```bash
# 1. 프로젝트 초기화
pip install moai-adk
moai-adk init

# 2. 프로젝트 설정
/alfred:0-project

# 3. SPEC 작성
/alfred:1-plan "새로운 기능 명세"

# 4. TDD 개발
/alfred:2-run SPEC-001

# 5. 동기화
/alfred:3-sync auto
```

## 🎯 핵심 원칙

### TRUST 5
- **Test First**: 항상 테스트부터 작성
- **Readable**: 명확하고 이해하기 쉬운 코드
- **Unified**: 일관된 개발 패턴
- **Secured**: 보안 우선 개발
- **Trackable**: 모든 변경 이력 추적

### 3단계 개발 워크플로우
1. **SPEC**: 요구사항 정의
2. **구축**: TDD 루프로 구현
3. **동기화**: 문서와 테스트 정렬
