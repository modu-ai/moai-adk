## 13. 수정 사항 상세 설명

### 13.1 좌측 서브 메뉴 스타일 수정

**파일**: `/Users/goos/MoAI/MoAI-ADK/docs/overrides/main.html`

**문제점**:
- 라이트 모드에서 서브 메뉴 배경색이 주 메뉴 배경과 동일하여 구분 불분명
- 다크 모드에서 서브 메뉴 배경색이 설정되지 않음
- 활성 메뉴 항목의 강조 효과가 불충분함

**해결책**:
1. **라이트 모드 서브 메뉴** (`.md-nav--secondary`)
   - 배경색: `#F2F1ED` (Bg Warm Off-White) → 명확한 구분
   - 텍스트색: `#504F4B` (Text Secondary) → 읽기 편함
   - 호버: `#EFEEEA` 배경 + `#171612` 텍스트 + 600 굵기
   - 활성: `#D5D4D0` 배경 + `#171612` 텍스트 + 700 굵기

2. **다크 모드 서브 메뉴** (`[data-md-color-scheme="slate"] .md-nav--secondary`)
