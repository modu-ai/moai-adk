## 📚 튜토리얼 시리즈

### 🚀 Hello World API (10분)

첫 번째 MoAI-ADK 프로젝트 - REST API 완성하기

**학습 목표**:

- MoAI-ADK 4단계 워크플로우 체험
- Alfred와의 상호작용 이해
- 완전한 TDD 개발 사이클 경험

**포함 주제**:

- SPEC 작성 (EARS 문법)
- RED-GREEN-REFACTOR 사이클
- Git 자동화
- 문서 동기화

**소요 시간**: 약 10분 **난이도**: 초급

## 💡 팁

각 튜토리얼은 다음 순서로 진행하는 것을 추천합니다:

1. **Hello World API** (10분) - 전체 워크플로우 체험
2. **기본 개념 학습** (20분) - 각 단계 이해
3. **SPEC 작성 연습** (30분) - 명세서 작성 능력
4. **실전 프로젝트** - 자신의 프로젝트에 적용
