# MoAI-ADK 문서 사이트 UI/UX 개선 - 빠른 시작 가이드

## 변경사항 한눈에 보기

### 1. 언어 선택자 (Language Selector)
- **위치**: 헤더 우측 (테마 전환 버튼 옆)
- **표시**: 이전 텍스트 레이블 → 현재 아이콘
  - 🇰🇷 한국어
  - 🌐 English
  - ⛩️ 日本語
  - 🏯 中文
- **기능**: 클릭하면 드롭다운 메뉴 표시, 언어 선택 가능

### 2. 색상 개선
| 요소 | 라이트 모드 | 다크 모드 |
|------|-----------|---------|
| 헤더 배경 | #14151A (검은색) | #171612 (어두운색) |
| 헤더 텍스트 | #FFFFFF (흰색) | #F2F1ED (밝은색) |
| 메뉴 텍스트 | #171612 (검은색) | #F2F1ED (밝은색) |
| 메뉴 배경 | #F7F6F2 (밝은색) | #1A1916 (어두운색) |

모든 텍스트/배경 대비가 **4.5:1 이상** (WCAG 2.1 AA 준수)

### 3. 반응형 디자인
- **모바일**: 언어 선택자가 아이콘만 표시
- **태블릿**: 아이콘 + 화살표 표시
- **데스크톱**: 아이콘 + 화살표 + 드롭다운 메뉴

## 기술 상세

### CSS 클래스 구조
```css
.md-select                  /* 컨테이너 */
├─ .md-select__inner        /* 버튼 */
│  ├─ .md-select__current-icon  /* 현재 언어 아이콘 */
│  └─ .md-select__arrow      /* 드롭다운 화살표 */
└─ .md-select__list         /* 드롭다운 리스트 */
   └─ .md-select__item      /* 각 언어 항목 */
      ├─ .md-select__icon    /* 언어 아이콘 */
      └─ .md-select__label   /* 언어 이름 */
```

### JavaScript 이벤트
```javascript
// 버튼 클릭 → 드롭다운 토글
button.click() → .md-select--active 클래스 토글

// 외부 클릭 → 드롭다운 닫기
document.click() → .md-select--active 클래스 제거

// 언어 선택 → 페이지 이동
.md-select__item.click() → window.location.href = lang.link
```

## 문제 해결

### 언어 선택자가 나타나지 않는 경우
1. 브라우저 캐시 삭제
2. 페이지 새로고침 (Ctrl+Shift+R 또는 Cmd+Shift+R)
3. 개발자 도구 (F12) → Console 에서 오류 확인

### 텍스트가 여전히 밝게 보이는 경우
1. 브라우저의 "독자 모드" 비활성화
2. 브라우저 줌 초기화 (Ctrl+0)
3. 다른 브라우저에서 테스트

### 아이콘이 글자로 보이는 경우
1. 브라우저 또는 OS에서 Emoji 지원 확인
2. 다른 디바이스에서 테스트
3. 개발자 도구에서 폰트 확인
