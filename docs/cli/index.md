# CLI Commands

MoAI-ADK CLI 명령어 가이드입니다.

## 목차

- [Commands](commands.md) - CLI 명령어 레퍼런스
- [Configuration](configuration.md) - 설정 파일 가이드

## 기본 명령어

```bash
# 프로젝트 초기화
moai init [project-name]

# 시스템 진단
moai doctor

# 프로젝트 상태 확인
moai status

# 버전 확인
moai version

# 도움말
moai help
```

## 설정 파일

- `.moai/config.json` - 프로젝트 설정
- `.moai/project/` - 프로젝트 정보
- `.moai/specs/` - SPEC 문서
- `.moai/memory/` - 개발 가이드
