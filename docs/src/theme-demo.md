# 뉴트럴 컬러 테마 데모

이 페이지는 MoAI-ADK 문서 사이트에 적용된 뉴트럴 컬러 팔레트를 보여주는 데모 페이지입니다.

## 🎨 적용된 컬러 팔레트

### 라이트 테마 컬러
- **Bg Warm Off-White**: `#F2F1ED` - 메인 배경색
- **Surface Very Light**: `#F7F6F2` - 카드 및 서피스 배경
- **Muted Surface**: `#EFEEEA` - 약간 어두운 서피스
- **Border Light Gray**: `#D5D4D0` - 테두리 및 구분선
- **Text Primary**: `#171612` - 주요 텍스트
- **Text Secondary**: `#504F4B` - 보조 텍스트
- **Neutral Mid**: `#C9C8C4` - 중간 톤 뉴트럴

### 다크 테마 컬러
- **Dark Background**: `#12110F` - 다크 메인 배경
- **Dark Surface**: `#1A1916` - 다크 서피스
- **Dark Surface Muted**: `#22211E` - 다크 뮤티드 서피스
- **Dark Border**: `#2E2D29` - 다크 테두리
- **Dark Text**: `#EEEDE8` - 다크 텍스트
- **Dark Accent**: `#F2F1ED` - 다크 테마 액센트

## 📄 텍스트 스타일링

이것은 **일반 텍스트**입니다. _이탤릭 텍스트_도 지원됩니다. `코드 인라인`은 다음과 같이 보입니다.

### 코드 블록 예시

```python
# 뉴트럴 테마 적용 예시
def apply_neutral_theme():
    colors = {
        'primary': '#171612',
        'secondary': '#504F4B',
        'background': '#F2F1ED',
        'surface': '#F7F6F2'
    }
    return colors
```

## 📋 표 스타일링

| 컬러 이름 | 라이트 테마 | 다크 테마 | 용도 |
|-----------|-------------|-----------|------|
| Primary | `#171612` | `#F2F1ED` | 주요 텍스트 및 액센트 |
| Secondary | `#504F4B` | `#C9C8C4` | 보조 텍스트 |
| Background | `#F2F1ED` | `#12110F` | 메인 배경 |
| Surface | `#F7F6F2` | `#1A1916` | 카드 및 서피스 |

!!! note "참고"
    이 뉴트럴 컬러 팔레트는 이미지에서 직접 추출된 색상을 기반으로 합니다.

!!! tip "팁"
    테마 전환 버튼을 클릭하여 라이트/다크 모드를 전환해보세요. 부드러운 애니메이션 효과가 적용됩니다.
