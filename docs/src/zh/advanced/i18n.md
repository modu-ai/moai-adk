# 多语言支持

MoAI-ADK 通过多语言文档系统为全球开发者提供最佳开发体验。

## 支持语言

- **한국어** (默认) - 完整支持
- **English** - 完整支持
- **日本語** - 完整支持
- **中文** - 完整支持

## 自动语言检测

根据用户的浏览器语言设置自动显示相应的语言版本。

## 文档结构

```
docs/src/
├── index.md           # 韩语（默认）
├── en/               # 英文文档
│   └── index.md
├── ja/               # 日语文档
│   └── index.md
└── zh/               # 中文文档
    └── index.md
```

## 翻译指南

1. **一致性**：所有技术术语使用统一翻译
2. **文化适应**：根据各语言文化特点调整表达方式
3. **更新同步**：新增功能时同步更新所有语言版本

## 贡献

如果您有兴趣添加新语言或改进现有翻译，请参考[贡献指南](../../contributing/index.md)。
