**State 1: Not Started** (0%)
- File structure exists
- Content is English template
- Needs translator

- Source updated

## Automation Tools

### Batch Translation Script

**File: `docs/scripts/translate_docs.py`**

```python
#!/usr/bin/env python3
"""
Batch translate documentation files using Claude API
"""

import os
from pathlib import Path
import anthropic

client = anthropic.Client(api_key=os.getenv("ANTHROPIC_API_KEY"))

def translate_file(source_path, target_lang):
    """Translate a single markdown file"""
    
    with open(source_path, 'r', encoding='utf-8') as f:
        content = f.read()
    
    prompt = f"""Translate the following markdown documentation from English to {target_lang}.

Rules:
1. Keep code blocks unchanged
2. Keep technical terms in English (SPEC, TDD, TAG, etc.)
3. Maintain markdown formatting
4. Preserve links and image paths
5. Use natural, professional tone

Content:
{content}
"""
    
    response = client.messages.create(
        model="claude-sonnet-4-5-20250929",
        max_tokens=8000,
        messages=[{
            "role": "user",
            "content": prompt
        }]
    )
    
    return response.content[0].text

def main():
    source_dir = Path("docs/src/en/")
    target_dir = Path("docs/src/es/")  # Change as needed
    
    for file_path in source_dir.rglob("*.md"):
        # Calculate target path
        relative_path = file_path.relative_to(source_dir)
        target_path = target_dir / relative_path
        
        # Skip if already translated
        if target_path.exists():
            print(f"âŠ˜ Skipping {relative_path} (already exists)")
            continue
        
        # Translate
        print(f"â†’ Translating {relative_path}...")
        translated = translate_file(file_path, "Spanish")
        
        # Write
        target_path.parent.mkdir(parents=True, exist_ok=True)
        with open(target_path, 'w', encoding='utf-8') as f:
            f.write(translated)
        
        print(f"âœ“ Saved to {target_path}")

if __name__ == '__main__':
    main()
```

### Translation Status Tracker

```python
#!/usr/bin/env python3
"""
Track translation progress across all languages
"""

import json
from pathlib import Path
from collections import defaultdict

def analyze_translations():
    """Analyze completion status for all languages"""
    
    base_dir = Path("docs/src/")
    languages = ['en', 'ko', 'ja', 'zh', 'es', 'fr', 'de']
    
    # Count files per language
    file_counts = defaultdict(int)
    for lang in languages:
        lang_dir = base_dir / lang
        if lang_dir.exists():
            file_counts[lang] = len(list(lang_dir.rglob("*.md")))
    
    # English is baseline (100%)
    baseline = file_counts['en']
    
    # Calculate completion
    status = {}
    for lang in languages:
        count = file_counts.get(lang, 0)
        percent = (count / baseline * 100) if baseline > 0 else 0
        status[lang] = {
            'files': count,
            'percent': round(percent, 1)
        }
    
    return status

def main():
    status = analyze_translations()
    
    print("Translation Status")
    print("â”" * 50)
    for lang, info in status.items():
        bar = "â–ˆ" * int(info['percent'] / 5)
        print(f"{lang:5} {bar:<20} {info['percent']:5.1f}% ({info['files']} files)")
    
    # Save JSON
    with open('docs/.moai/reports/translation-status.json', 'w') as f:
        json.dump(status, f, indent=2)

if __name__ == '__main__':
    main()
```

## Community Translation

### Becoming a Translator

**Step 1: Choose a Language**

Check [Translation Status](https://github.com/modu-ai/moai-adk/issues?q=label:translation) for available languages.

**Step 2: Claim a Section**

```markdown
Comment on GitHub issue:

I would like to translate the following sections to Spanish:
- Getting Started (10 files)
- Alfred Guide (5 files)

Estimated completion: 2 weeks
```

**Step 3: Translate**

Follow the [Translation Workflow](#translation-workflow).

**Step 4: Submit PR**

```bash
git checkout -b docs/spanish-getting-started
git add docs/src/es/getting-started/
git commit -m "docs: Translate Getting Started to Spanish"
gh pr create --title "Spanish: Getting Started section"
```

### Translation Guidelines

**Do**:
- Use glossary for consistency
- Maintain markdown formatting
- Keep code examples in English
- Test links after translation
- Follow style guide for target language

**Don't**:
- Use machine translation without review
- Change technical meanings
- Add/remove content arbitrarily
- Mix languages within sentences
- Ignore cultural context

### Recognition

Contributors are recognized in:
- Documentation contributors page
- Release notes
- README.md acknowledgments
- Special translator badge (GitHub)
