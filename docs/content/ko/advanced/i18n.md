# 다국어 지원

MoAI-ADK는 다국어 문서 시스템을 통해 전 세계 개발자에게 최고의 개발 경험을 제공합니다.

## 지원 언어

- **한국어** (기본) - 완벽 지원
- **English** - 완벽 지원
- **日本語** - 완벽 지원
- **中文** - 완벽 지원

## 자동 언어 감지

사용자의 브라우저 언어 설정에 따라 자동으로 적절한 언어 버전을 표시합니다.

## 문서 구조

```
docs/src/
├── index.md           # 한국어 (기본)
├── en/               # 영어 문서
│   └── index.md
├── ja/               # 일본어 문서
│   └── index.md
└── zh/               # 중국어 문서
    └── index.md
```

## 번역 가이드라인

1. **일관성**: 모든 기술 용어는 일관된 번역을 사용
2. **문화적 적응**: 각 언어의 문화적 특성에 맞게 표현 조정
3. **업데이트 동기화**: 새로운 기능 추가 시 모든 언어 동시 업데이트

## 기여하기

새로운 언어 추가나 기존 번역 개선에 관심이 있으시면 [기여 가이드](../contributing/index.md)를 참조하세요.
