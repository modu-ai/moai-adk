# MoAI-ADK 온라인 문서 목차 구조 제안

> **기반 분석**: Claude 플랫폼 공식 문서 및 Claude Code 공식 문서 구조 분석
> **대상**: 엔터프라이즈급 AI 개발 키트(MoAI-ADK) 사용자
> **목표**: 실용적이고 실행 가능한 개발자 가이드 제공

## 📋 분석 요약

### Claude 플랫폼 문서 특징
- **구조**: 기능별 카테고리화 (API, Agents, Tools, Build with Claude)
- **강점**: 실제 코드 예시, 프롬프트 엔지니어링 베스트 프랙티스, 단계별 가이드
- **패턴**: Quick Reference → Implementation Guide → Advanced Documentation

### Claude Code 문서 특징
- **구조**: 사용 시나리오 기반 (Overview, Quickstart, Common Workflows, Integrations)
- **강점**: 실용적인 워크플로우, Skills/Commands/Agents 개발 가이드
- **패턴**: 시작 가이드 → 고급 기능 → 통합 패턴

### MoAI-ADK 특징 분석
- **핵심**: Alfred 에이전트 오케스트레이션, SPEC 기반 개발, TDD 워크플로우
- **구성요소**: 24개 활성 에이전트, 5계층 구조, 다국어 지원
- **강점**: 자동화된 개발 파이프라인, 품질 관리, 문서 관리 시스템
