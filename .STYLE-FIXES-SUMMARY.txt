2. 해결된 문제
================================================================================

문제 1: 라이트 모드 서브 메뉴 배경색이 주 메뉴와 동일
→ 해결: #F2F1ED (Bg Warm Off-White)로 설정 → 명확한 구분

문제 2: 다크 모드 서브 메뉴 배경색 미설정
→ 해결: #1A1916 (Dark Surface Very Light)로 설정 → 테마 적용

문제 3: 활성 메뉴 항목 강조 효과 불충분
→ 해결: 배경색 변경 (#D5D4D0 라이트 / #3A3935 다크) + 폰트 굵기 700

6. 배포 지침
================================================================================

1단계: 파일 변경사항 확인
  $ git diff docs/overrides/main.html
  $ git diff docs/stylesheets/extra.css

2단계: MkDocs 빌드 및 로컬 테스트
  $ cd /Users/goos/MoAI/MoAI-ADK/docs
  $ mkdocs build
  $ mkdocs serve

3단계: 브라우저에서 테스트
  • http://localhost:8000 (기본 포트) 또는 9999 (설정된 포트)
  • 라이트 모드: 모든 메뉴 항목 색상 확인
  • 다크 모드: 테마 토글 후 색상 확인
  • 호버 효과: 마우스 오버 시 배경/텍스트 변화 확인
  • 활성 상태: 현재 페이지 메뉴 하이라이트 확인

4단계: 커밋 및 푸시
  $ git add docs/overrides/main.html docs/stylesheets/extra.css
  $ git commit -m "style: Fix left sidebar submenu styling for light/dark themes"
  $ git push origin develop
