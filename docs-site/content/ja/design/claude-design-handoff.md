# Claude Designハンドオフ

## Claude Designの概要

**Claude Design**は、Anthropicが2026年4月17日にリリースしたAIベースの設計生成ツールです。Claude.ai内で自然言語でUI/UX設計を生成できます。

- **基盤モデル:** Claude 3.5 Opus
- **アクセス:** https://claude.ai/design
- **出力形式:** 設計トークン、コンポーネント仕様、静的資産、ハンドオフバンドル

## サポートされるサブスクリプションプラン

| プラン | Claude Designサポート | 備考 |
|---|---|---|
| Free | 未サポート | Claude.ai/designにアクセス不可 |
| Pro | サポート | $20/月 |
| Max | サポート | $200/月 |
| Team | サポート(デフォルト: 管理者オフ) | チーム単位で請求 |
| Enterprise | サポート(デフォルト: 管理者オフ) | 契約ベース |

**注意:** TeamおよびEnterpriseプランは、管理者がデフォルトで機能を無効化します。有効化をチーム管理者にリクエストしてください。

## サポート入力形式

| 形式 | 説明 |
|---|---|
| **テキスト** | 自然言語で設計要件を記述 |
| **画像** | 参考デザイン案をアップロード |
| **DOCX/PPTX** | 既存ドキュメントまたはプレゼンテーション |
| **XLSX** | データテーブルと構造化情報 |
| **ウェブキャプチャ** | URLからウェブサイトスクリーンショット |
| **Figma** | Figmaファイルとフレーム |
| **GitHub** | リポジトリコードとREADME |

## ハンドオフバンドルエクスポート

### ステップ1: Claude.ai/designにアクセス

ブラウザで**https://claude.ai/design**を開く

### ステップ2: デザイン作成

Claude Designインターフェースで:
- 自然言語でデザイン説明を入力
- 参考画像/ドキュメントをアップロード
- リアルタイムUIプレビュー生成

### ステップ3: バンドルエクスポート

Claude DesignのExportまたはShareメニューから:
- **ZIP形式:** すべての設計ファイル、トークン、資産を含む
- **PDF形式:** 静的ドキュメント版(選択)
- **Canva/Figma形式:** 外部ツールで継続編集(選択)

**推奨:** ZIP形式でエクスポート

### ステップ4: ローカル保存

エクスポートファイルをローカルファイルシステムに保存:

```bash
# 例: ~/Downloads/my-design.zip
```

## MoAI-ADKへのバンドルインポート

### ステップ5: /moai designを再実行

Claude Codeで:

```
/moai design
```

パスA(Claude Design)選択後:

```
バンドルパスを入力してください: ~/Downloads/my-design.zip
```

### ステップ6: 自動変換

`moai-workflow-design-import`スキルが:
- バンドルを解析
- 設計トークンをJSONに変換
- コンポーネント仕様を抽出
- 静的資産をコピー

結果ファイル:
```
.moai/design/
├── tokens.json          # 設計トークン
├── components.json      # コンポーネント仕様
└── assets/              # 画像、アイコン
```

## エラーコード

バンドルインポート失敗時:

| エラーコード | 原因 | 解決策 |
|---|---|---|
| `DESIGN_IMPORT_NOT_FOUND` | ファイルが見つからない | パス確認、ファイル存在確認 |
| `UNSUPPORTED_FORMAT` | ZIP以外の形式 | ZIP形式で再エクスポート |
| `UNSUPPORTED_VERSION` | サポートされていないバージョン | 最新版で再エクスポート |
| `SECURITY_REJECT` | セキュリティチェック失敗 | 管理者に連絡 |
| `MISSING_MANIFEST` | バンドル構造破損 | 新しいバンドル生成して再試行 |

## フォールバックパス

Claude Designが利用できない場合またはバンドルインポートに失敗した場合:

### オプション1: パスBに切り替え

```
バンドルインポート失敗。パスBに切り替えますか?
→ [はい] [いいえ]
```

選択するとコードベース設計ワークフローに自動切り替え

### オプション2: バンドルを再生成

Claude.ai/designに戻り:
1. 既存デザイン修正または新規作成
2. ExportメニューからZIPを再生成
3. 新しいファイルで`/moai design`を再実行

## チームコラボレーション

Claude DesignのTeamサブスクリプション機能:

- **リアルタイムコラボレーション:** 複数チームメンバーが同時にデザイン編集
- **共有リンク:** チーム外に設計を共有(読み取り専用)
- **バージョン履歴:** 以前のバージョンを復元可能
- **コメント:** 設計へのフィードバック記録

**注意:** デフォルト無効のためチーム管理者が有効化必要
