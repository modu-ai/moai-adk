# 빠른 시작

## 5분 안에 시작하기

### Step 1: 설치 (1분)

```bash
uv tool install moai-adk
moai-adk --version
```

### Step 2: 프로젝트 생성 (1분)

```bash
moai-adk init hello-world
cd hello-world
```

### Step 3: Alfred 시작 (3분)

```bash
claude
```

Claude Code에서:

```
/alfred:0-project
```

Alfred가 물어볼 것입니다:

1. **프로젝트 이름?** → hello-world
2. **프로젝트 목표?** → Learning MoAI-ADK
3. **주 개발 언어?** → python
4. **모드?** → personal

## 첫 번째 기능 구현

### Step 1: SPEC 작성

```
/alfred:1-plan "사용자 인증 기능"
```

Alfred가 SPEC을 작성합니다:
- `.moai/specs/SPEC-AUTH-001/spec.md`
- 명확한 EARS 요구사항
- @TAG 체인 생성

### Step 2: TDD 구현

```
/alfred:2-run SPEC-AUTH-001
```

Alfred가 자동으로:
1. **RED**: 테스트 작성
2. **GREEN**: 구현
3. **REFACTOR**: 코드 개선

### Step 3: 문서 동기화

```
/alfred:3-sync
```

Alfred가 업데이트:
- README.md
- CHANGELOG.md
- API 문서
