# PART 1용 템플릿: basic-tutorial (기초 튜토리얼)

> **대상**: 초급 학습자
> **특성**: 친근하고 격려하는 톤, 단계별 체크포인트, 자주 "왜?" 질문에 답변
> **문체**: 존댓말 (합니다체), 대화형
> **예상 글자 수**: 1500-2000자 (초급자 집중력 고려)
> **적용 PART**: PART 1 - Claude Code 기초 (Chapter 1-5)

---

## 📌 이 템플릿의 목적

초급 학습자가 **처음 마주하는 개념**을 쉽게 이해할 수 있도록:
- 실생활 비유로 개념 도입
- 단계별 체크포인트로 성취감 제공
- "왜?"에 대한 명확한 답변
- 짧은 문장과 명확한 구조

---

## 템플릿 구조

### 1. 이전 장 연결 (2-3문장)

```markdown
## 📖 이전 장 복습

[이전 장 제목]에서는 **[핵심 개념 1-2개]**를 배웠습니다.

이제 이 개념을 바탕으로 **[현재 장의 주제]**를 알아보겠습니다.

> 💡 **잠깐!** 이전 장의 [개념]이 기억나지 않는다면, [N-1]장의 "[섹션명]"을 다시 읽어보세요.
```

**가이드**:
- 이전 장의 `chapter-summaries/{n-1:02d}-*.json`에서 key_concepts 확인
- 중복 설명 금지: "앞서 배운..." 표현 사용
- 용어 일관성: 이전 장의 terminology 맵 참고

### 2. 학습 목표 (3-5개, 명확한 행동 동사)

```markdown

이 장을 마치면 다음을 할 수 있습니다:

- [ ] [개념]을 **이해**하고 실생활 예시로 **설명**할 수 있습니다
- [ ] [도구/방법]을 사용하여 [작업]을 **수행**할 수 있습니다
- [ ] [A]와 [B]의 차이점을 **구별**할 수 있습니다
- [ ] 간단한 [실습 프로젝트]를 **스스로 완성**할 수 있습니다
```

**가이드**:

### 요리 레시피를 따라 하는 것과 비슷해요

음식점에 가면 주방장이 요리를 만들어 주죠? 여러분은 메뉴를 고르고, 주방장은 요리를 만듭니다.

Claude Code도 마찬가지입니다! 여러분이 "무엇을 만들고 싶은지" 말하면, Claude가 코드를 만들어 줍니다.

### 프로그래밍에서도 마찬가지입니다

전통적인 코딩에서는 여러분이 직접 모든 코드를 작성해야 했습니다.

하지만 Claude Code를 사용하면:
- 여러분: "로그인 기능 만들어 줘"
- Claude: "네, 이렇게 만들었어요! [코드]"

> 🤔 **잠깐 생각해 보세요**: 여러분이 만들고 싶은 프로그램이 있나요?

## 1.1 Claude Code란?

### 📖 Claude Code란?

**간단히 말하면**: 터미널에서 실행되는 AI 코딩 비서입니다.

Claude Code는 Anthropic이 만든 AI 도구로, 여러분의 터미널 안에서 코드를 작성하고, 파일을 수정하고, 명령어를 실행할 수 있습니다.

### 🤔 왜 필요한가요?

**기존 방법의 문제점**:
- ❌ 모든 코드를 직접 작성하느라 시간이 오래 걸림
- ❌ 문법 오류를 찾느라 디버깅에 많은 시간 소비
- ❌ 새로운 라이브러리를 배우는 데 시간이 너무 많이 듦

**Claude Code의 장점**:
- ✅ AI가 코드를 자동으로 생성 → 개발 시간 50% 단축
- ✅ 실시간 오류 감지 및 수정 제안 → 디버깅 시간 70% 감소
- ✅ 공식 문서 자동 조회 (Context7 MCP) → 학습 시간 단축

> 💡 **핵심**: Claude Code는 여러분의 개발 속도를 2-3배 높여주는 AI 비서입니다!

### 👉 어떻게 사용하나요?

**Step 1: 터미널 열기**

맥/리눅스:
```bash
# Terminal 앱 실행 (Cmd + Space → "Terminal" 입력)
```

윈도우:
```bash
# PowerShell 실행 (Win + X → "PowerShell" 선택)
```

✅ **체크포인트**: 터미널이 열렸나요? 검은색/흰색 화면이 보이면 성공!

**Step 2: Claude 실행**

```bash
# Claude Code 실행
claude

# 결과:
# Welcome to Claude Code!
# How can I help you today?
```

✅ **체크포인트**: "Welcome to Claude Code!" 메시지가 보이면 성공!

**Step 3: 첫 질문하기**

```
You: "안녕, Claude! 파이썬으로 'Hello, World!'를 출력하는 코드 만들어 줘"
```

Claude가 자동으로 코드를 생성합니다:

```python
# hello.py
print("Hello, World!")
```

> ⚠️ **주의**: 첫 실행 시 API 키 설정이 필요합니다. (1.3 섹션 참조)
