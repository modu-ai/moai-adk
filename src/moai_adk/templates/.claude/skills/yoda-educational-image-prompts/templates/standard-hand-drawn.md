## 기본 템플릿 구조

```
핸드 드로잉 삽화 스타일의 '[주제/제목]' 인포그래픽입니다.
종이 질감 배경에 색연필로 그린 듯한 따뜻한 색감과 손으로 그린 선,
유기적인 화살표가 특징입니다. 손글씨로 된 제목과 [구체적 설명],
그리고 [요소들] 그림이 포함되어 있습니다.

주요 키워드 및 요소:
- 스타일: 핸드 드로잉, 스케치 노트, 만화, 따뜻한 색연필 질감
- 배경: 종이 질감 (크림색 또는 베이지)
- 텍스트: 손글씨 스타일
- 요소: 손으로 그린 [구체적 요소], 유기적인 화살표 및 선
- 전반적인 느낌: 친근하고 따뜻하며 정보 전달력이 높은 인포그래픽
```

---

## 사용 규칙

### ✅ 필수 포함 요소

1. **"핸드 드로잉 삽화 스타일"** - 항상 첫 문장에 명시
2. **"종이 질감 배경"** - 크림색 또는 베이지 지정
3. **"색연필로 그린 듯한"** - 따뜻한 색감 강조
4. **"손으로 그린 선"** - 손글씨 스타일 명시
5. **"유기적인 화살표"** - 연결 강조
6. **"주요 키워드 및 요소"** - 하단에 구조화된 리스트

### ❌ 피해야 할 표현

- "실제 스크린샷", "고해상도", "전문적인" (기계적 느낌)
- "디지털 디자인", "벡터 그래픽" (손그림 느낌 손실)
- "깔끔한", "정돈된" (너무 정형화됨)
- 구체적인 색상 코드 (#FF0000 등) → 대신 "따뜻한 주황색" 사용

## 콘텐츠 타입별 예시

### 1. 개념 비교 (Before-After)

```
핸드 드로잉 삽화 스타일의 '전통적 코딩 vs 에이전틱 코딩' 비교 인포그래픽입니다.
종이 질감 배경에 색연필로 그린 듯한 따뜻한 색감과 손으로 그린 선,
유기적인 화살표가 특징입니다.
왼쪽에는 개발자가 여러 작업을 동시에 수행하는 모습(전통적 코딩),
오른쪽에는 AI 에이전트들과 협업하는 개발자 모습(에이전틱 코딩)이
손글씨 라벨과 함께 표현되어 있습니다.

주요 키워드 및 요소:
- 스타일: 핸드 드로잉, 스케치 노트, 만화, 따뜻한 색연필 질감
- 배경: 종이 질감 (크림색)
- 텍스트: 손글씨 스타일 라벨
- 요소: 손으로 그린 사람, 로봇, 태스크 아이콘, 유기적인 화살표
- 전반적인 느낌: 친근하고 따뜻하며 비교가 명확한 인포그래픽
```

### 2. 프로세스 플로우

```
핸드 드로잉 삽화 스타일의 'Claude Code 작업 사이클' 순환 다이어그램입니다.
종이 질감 배경에 색연필로 그린 듯한 따뜻한 색감과 손으로 그린 선,
유기적인 화살표가 특징입니다.
중앙에 순환 화살표로 연결된 5단계(요청→분석→생성→실행→검증)가
손글씨로 표시되어 있고, 각 단계마다 간단한 아이콘 그림이 포함되어 있습니다.

주요 키워드 및 요소:
- 스타일: 핸드 드로잉, 스케치 노트, 만화, 따뜻한 색연필 질감
- 배경: 종이 질감 (베이지)
- 텍스트: 손글씨 스타일 단계 설명
- 요소: 손으로 그린 순환 화살표, 단계별 아이콘, 숫자 라벨
- 전반적인 느낌: 친근하고 따뜻하며 흐름이 명확한 인포그래픽
```

### 3. 핵심 역량 (다중 요소)

```
핸드 드로잉 삽화 스타일의 '에이전틱 개발자 5가지 핵심 역량' 인포그래픽입니다.
종이 질감 배경에 색연필로 그린 듯한 따뜻한 색감과 손으로 그린 선,
유기적인 화살표가 특징입니다.
중앙에 개발자 아이콘이 있고, 주변 5개 섹션(문제정의, 아키텍처, 프롬프트, 리뷰, 통합)이
손글씨 라벨과 함께 별 모양 레이아웃으로 배치되어 있습니다.

주요 키워드 및 요소:
- 스타일: 핸드 드로잉, 스케치 노트, 만화, 따뜻한 색연필 질감
- 배경: 종이 질감 (크림색)
- 텍스트: 손글씨 스타일 역량 설명
- 요소: 손으로 그린 사람, 5개 섹션 아이콘, 유기적인 연결선
- 전반적인 느낌: 친근하고 따뜻하며 구조가 명확한 인포그래픽
```

### 4. 터미널/코드 화면

```
핸드 드로잉 삽화 스타일의 '터미널 명령어 실행 화면' 일러스트입니다.
종이 질감 배경에 색연필로 그린 듯한 따뜻한 색감과 손으로 그린 선이 특징입니다.
손으로 그린 터미널 창 프레임 안에 명령어와 실행 결과가 손글씨로 표시되어 있고,
중요한 부분에는 손으로 그린 화살표와 설명 라벨이 추가되어 있습니다.

주요 키워드 및 요소:
- 스타일: 핸드 드로잉, 스케치 노트, 따뜻한 색연필 질감
- 배경: 종이 질감 (베이지)
- 텍스트: 손글씨 스타일 명령어와 출력
- 요소: 손으로 그린 터미널 창, 커서, 화살표 주석
- 전반적인 느낌: 친근하고 학습하기 쉬운 터미널 화면
```

## 자동 생성 함수 (개념)

```python
def generate_hand_drawn_prompt(
    title: str,
    content_type: str,  # "comparison", "flow", "multi-element", "screen"
    elements: list[str],
    description: str
) -> str:
    """
    표준 핸드 드로잉 프롬프트 자동 생성
    """
    base = f"핸드 드로잉 삽화 스타일의 '{title}' 인포그래픽입니다.\n"
    base += "종이 질감 배경에 색연필로 그린 듯한 따뜻한 색감과 손으로 그린 선, \n"
    base += "유기적인 화살표가 특징입니다. "
    base += f"{description}\n\n"

    base += "주요 키워드 및 요소:\n"
    base += "- 스타일: 핸드 드로잉, 스케치 노트, 만화, 따뜻한 색연필 질감\n"
    base += "- 배경: 종이 질감 (크림색 또는 베이지)\n"
    base += "- 텍스트: 손글씨 스타일\n"
    base += f"- 요소: 손으로 그린 {', '.join(elements)}, 유기적인 화살표\n"
    base += "- 전반적인 느낌: 친근하고 따뜻하며 정보 전달력이 높은 인포그래픽"

    return base
```

## 이미지 개수 가이드

### 챕터당 권장 이미지 개수

| 섹션 길이 | 필수 이미지 | 선택 이미지 | 총 개수 |
|---------|----------|----------|--------|
| 5 pages | 1-2 | 0-1 | 1-3 |
| 7 pages | 2-3 | 0-1 | 2-4 |
| 10 pages | 2-3 | 1-2 | 3-5 |
| 15+ pages | 3-4 | 1-2 | 4-6 |

### 이미지 선택 우선순위

1. **필수 (Priority 1)**: 핵심 개념 소개, Before-After 비교
2. **권장 (Priority 2)**: 프로세스 플로우, 구조 다이어그램
3. **선택 (Priority 3)**: 터미널 화면, 예제 결과

## Mermaid 대체 가이드

다음 경우에는 이미지 대신 Mermaid 다이어그램 사용:

### ✅ Mermaid 사용 권장

- 간단한 플로우차트 (3-5단계)
- 시퀀스 다이어그램 (클래스 간 상호작용)
- 상태 다이어그램 (상태 전이)
- 간단한 클래스 다이어그램

### ❌ 이미지 사용 권장

- 복잡한 아키텍처 (6개 이상 컴포넌트)
- 감정/분위기 전달이 필요한 경우
- Before-After 비교
- 터미널/코드 화면 예시
