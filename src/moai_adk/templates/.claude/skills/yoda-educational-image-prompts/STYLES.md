## 🎨 Style 1: 핸드드로잉 스케치

### 설명

따뜻하고 친근한 손그림 스타일. 교육용 화이트보드에서 그린 것 같은 자연스러운 느낌입니다.

**최적 용도**:
- 개념 소개 및 개괄 설명
- 첫 강의 슬라이드 (Hero 이미지)
- 학습자들이 처음 접하는 개념
- 워크숍 및 브레인스토밍 자료

**특징**:
- 색연필 또는 마커 질감
- 손으로 그린 듯한 유기적인 선
- 종이 또는 화이트보드 배경
- 손글씨 라벨과 주석
- 유기적인 화살표와 연결선

### 한국어 프롬프트 템플릿

```
"{주제}을(를) 표현한 핸드드로잉 삽화 스타일입니다.
{배경} 배경에 {색상1}과(와) {색상2} 색연필로 그린 듯한 따뜻한 색감,
손으로 그린 유기적인 화살표가 {설명}을(를) 표시합니다.
{요소1}, {요소2}, {요소3} 등 주요 내용이 손글씨로 표기되어 있으며,
각 항목을 나타내는 아이콘 그림이 포함되어 있습니다.
{분위기} 느낌의 친근하고 접근하기 쉬운 디자인입니다.
명확한 한글 서체와 높은 가독성, 교육용 콘텐츠를 위한 적절한 대비."
```

### 구체적 예제

#### 예제 1.1: React Hooks 소개

```
"React Hooks를 표현한 핸드드로잉 삽화 스타일입니다.
크림색 종이 질감 배경에 주황색과 파란색 색연필로 그린 듯한 따뜻한 색감,
손으로 그린 유기적인 화살표가 함수 컴포넌트에서 Hooks로의 흐름을 표시합니다.
'useState', 'useEffect', 'useContext' 등 주요 Hooks가 손글씨로 표기되어 있으며,
각 Hook을 나타내는 아이콘 그림이 포함되어 있습니다.
교육용 화이트보드 느낌의 친근하고 접근하기 쉬운 디자인입니다.
명확한 한글 서체와 높은 가독성, 교육용 콘텐츠를 위한 적절한 대비."
```

#### 예제 1.2: 파이썬 리스트 기초

```
"파이썬 리스트를 표현한 핸드드로잉 삽화 스타일입니다.
흰색 종이 질감 배경에 검은 마커와 다채로운 색연필로 그린 듯한 생동감 있는 색감,
손으로 그린 유기적인 괄호와 화살표가 인덱싱과 요소 접근을 표시합니다.
'list[0]', 'list[1]', 'list[-1]' 등 인덱싱 표기법이 손글씨로 표기되어 있으며,
각 위치의 값을 나타내는 작은 박스 그림이 포함되어 있습니다.
교과서의 여백에 필기하듯이 설명하는 친근한 느낌의 디자인입니다.
명확한 한글 서체와 적절한 여백, 학습자 친화적 가독성."
```

#### 예제 1.3: 데이터베이스 정규화 개념

```
"데이터베이스 정규화 개념을 표현한 핸드드로잉 삽화 스타일입니다.
노란 종이 배경에 검은색 펜과 빨간색, 파란색 형광펜으로 그린 강의 노트 느낌,
손으로 그린 곡선 화살표가 비정규형 → 1NF → 2NF → 3NF 진행 과정을 표시합니다.
각 정규화 단계가 손글씨로 크게 표기되어 있으며,
테이블과 열의 분해 과정을 나타내는 간단한 박스 다이어그램이 포함되어 있습니다.
강의실 칠판에 실시간으로 설명하는 것처럼 생생하고 단계별인 디자인입니다.
명확한 한글 서체, 충분한 여백, 그룹화된 색상 사용."
```

## 🎨 Style 2: 아이소메트릭 3D 기술도

### 설명

30도 각도로 깊이감을 표현한 평면 3D 디자인. 기술적이고 전문적인 느낌입니다.

**최적 용도**:
- 시스템 아키텍처 및 구조도
- 마이크로서비스 설명
- 네트워크 토폴로지
- 기술 컴포넌트 관계도
- 고급 개념 설명

**특징**:
- 30도 각도의 정확한 기하학
- 평면 색상 (그라데이션 최소)
- 미세한 그림자로 깊이감 표현
- 정밀한 기술 선
- 기술용 글꼴

### 한국어 프롬프트 템플릿

```
"{주제}의 아이소메트릭 3D 삽화입니다.
30도 각도로 깊이감을 표현한 평면 디자인 스타일이며,
{색상} 계열의 {개수}가지 색상 팔레트를 사용합니다.
{요소1}, {요소2}, {요소3}가 정밀한 형태로 디자인되고,
{연결} 관계가 기하학적인 선으로 명확하게 표현됩니다.
깊이감을 위한 미세한 그림자와 선 두께의 변화가 있으며,
각 요소는 명확하게 구분되는 색상으로 표시됩니다.
전문적인 기술 다이어그램 스타일이며,
명확한 한글 라벨과 높은 가독성을 갖춘 설계입니다."
```

### 구체적 예제

#### 예제 2.1: 마이크로서비스 아키텍처

```
"마이크로서비스 아키텍처의 아이소메트릭 3D 삽화입니다.
컨테이너, 데이터베이스, API 게이트웨이, 메시지 큐가 30도 각도로 배치되어 입체감을 표현하며,
깔끔한 파란색, 회색, 녹색 계열의 색상 팔레트를 사용합니다.
각 마이크로서비스는 정밀한 큐브 형태로 디자인되고,
서비스 간 통신은 기하학적인 선과 화살표로 명확하게 표현됩니다.
깊이감을 위한 미세한 그림자와 선 두께 변화가 있으며,
'주문 서비스', '결제 서비스', '알림 서비스' 등 한글 라벨이 명확하게 표시됩니다.
IT 전문가를 위한 기술 다이어그램 스타일입니다.
고해상도 교육용 삽화로 적합하며, 명확한 한글 타이포그래피와 높은 대비를 갖춥니다."
```

#### 예제 2.2: 클라우드 인프라 레이아웃

```
"AWS 클라우드 인프라 구성의 아이소메트릭 3D 삽화입니다.
EC2 인스턴스, RDS 데이터베이스, S3 버킷, 로드 밸런서가 30도 각도로 배치되어 공간감을 표현하며,
주황색(AWS), 파란색(서비스), 회색(인프라) 계열의 색상 팔레트를 사용합니다.
각 AWS 서비스는 특징적인 형태로 디자인되고,
VPC, 서브넷, 보안 그룹 같은 네트워킹 구조가 명확하게 표현됩니다.
깊이감을 위한 정밀한 그림자와 입체 효과,
'가용성 영역 A', '가용성 영역 B' 등 한글 라벨이 각 영역에 표시됩니다.
클라우드 아키텍처 설계 교육용 전문 다이어그램입니다."
```

#### 예제 2.3: 메모리 구조 (Stack vs Heap)

```
"컴퓨터 메모리 구조(Stack vs Heap)의 아이소메트릭 3D 삽화입니다.
Stack 메모리와 Heap 메모리가 30도 각도로 분리되어 표현되며,
파란색(Stack), 녹색(Heap), 회색(OS) 계열 색상을 사용합니다.
Stack 영역의 정렬된 메모리 셀과 Heap 영역의 동적 할당 영역이 명확하게 구분되고,
포인터와 메모리 주소를 나타내는 화살표가 기하학적으로 연결됩니다.
각 영역의 경계, 성장 방향, 메모리 해제 과정이 시각적으로 표현되며,
한글 라벨로 각 구조의 특징이 설명됩니다.
프로그래밍 초급자도 이해하기 쉬운 기술 교육용 다이어그램입니다."
```

## 🎨 Style 3: 미니멀 플랫 디자인

### 설명

단순한 형태와 명확한 색상으로 정보를 직관적으로 전달하는 스타일입니다.

**최적 용도**:
- 프로세스 흐름도 및 워크플로우
- UI/UX 개념 설명
- 단계별 튜토리얼
- 비교 및 대조
- 초급 학습자용 개념도

**특징**:
- 그라데이션 없음 (단색만 사용)
- 단순한 기하학적 형태
- 높은 대비의 색상
- 깔끔한 산세리프 폰트
- 최소한의 장식

### 한국어 프롬프트 템플릿

```
"{주제}의 미니멀 플랫 디자인 삽화입니다.
그라데이션이나 그림자 없이 {색상1}, {색상2}, {색상3} 등 명확한 색상으로만 표현하며,
단순한 기하학적 도형(원, 사각형, 삼각형)을 사용합니다.
{요소1}, {요소2}, {요소3}이(가) 명확하게 구분되는 색상 블록으로 표현되고,
{연결 방식} 방식으로 요소 간의 관계가 표시됩니다.
명확성을 위한 대담한 색상 대비와 깔끔한 산세리프 라벨,
현대적인 디지털 디자인 미학이 특징입니다.
모든 연령과 배경의 학습자가 직관적으로 이해할 수 있는 설계입니다."
```

### 구체적 예제

#### 예제 3.1: 인증 플로우 (Authentication Flow)

```
"웹 애플리케이션 인증 플로우의 미니멀 플랫 디자인 삽화입니다.
그라데이션이나 그림자 없이 파란색(사용자), 초록색(서버), 주황색(토큰) 세 가지 색상으로 표현하며,
동그란 아이콘과 사각형 박스만을 사용합니다.
사용자 아이콘, 로그인 폼, 서버, 데이터베이스가 명확하게 구분되는 색 블록으로 표현되고,
'로그인 입력 → 서버 검증 → 토큰 발급 → 인증 완료' 순서가 굵은 화살표로 표시됩니다.
명확한 한글 라벨과 매우 높은 색상 대비, 깔끔한 산세리프 폰트가 사용됩니다.
학습자들이 한 눈에 인증 과정을 이해할 수 있는 명료한 설계입니다."
```

#### 예제 3.2: Git 커밋 흐름

```
"Git 커밋 과정의 미니멀 플랫 디자인 삽화입니다.
그라데이션 없이 보라색(변경사항), 파란색(스테이징), 초록색(커밋) 세 가지 색상만 사용하며,
동그란 아이콘과 직각 사각형만으로 표현됩니다.
'작업 영역 → 스테이징 영역 → 저장소' 세 가지 상태가 색상이 다른 박스로 표시되고,
파일 아이콘이 각 단계를 거치는 과정이 분명한 화살표로 표현됩니다.
'git add', 'git commit', 'git push' 한글 명령어가 각 화살표 위에 명확하게 표시되며,
최대한 단순하면서도 전체 과정이 명확하게 보이는 설계입니다.
초급 개발자도 Git의 기본 개념을 쉽게 학습할 수 있습니다."
```

#### 예제 3.3: API 요청-응답 사이클

```
"REST API 요청-응답 사이클의 미니멀 플랫 디자인 삽화입니다.
그라데이션 없이 파란색(클라이언트), 초록색(서버), 노란색(데이터) 색상으로만 표현하며,
컴퓨터, 서버, 데이터베이스 세 가지 단순한 아이콘을 사용합니다.
클라이언트 → 서버로의 요청(파란 화살표)과 서버 → 클라이언트로의 응답(초록 화살표)이 명확하게 표시되고,
각 단계에서 주고받는 데이터가 노란색 데이터 아이콘으로 표현됩니다.
'GET /users', '200 OK', 'JSON 데이터' 등이 한글과 함께 명확하게 라벨링되며,
최소한의 시각 요소로 최대한의 정보를 전달하는 현대적 설계입니다."
```

<a name="style-4"></a>
## 🎨 Style 4: 기술 도면 블루프린트

### 설명

엔지니어링 도면이나 건축 도면 같은 기술 도면 스타일입니다.

**최적 용도**:
- 알고리즘 상세 설명
- 데이터 구조 내부 동작
- 메모리 레이아웃
- 시스템 상세 사양
- 기술 스펙 문서

**특징**:
- 격자 배경 또는 청사진 질감
- 정밀한 측정값과 주석
- 단색(검은색/파란색) 선
- 기술 및 과학적 느낌
- 정확한 비율과 각도

### 한국어 프롬프트 템플릿

```
"{주제}의 기술 도면 블루프린트 스타일 삽화입니다.
측정값과 상세한 주석이 포함된 정밀한 엔지니어링 다이어그램이며,
격자 배경 또는 청사진 용지 질감을 배경으로 사용합니다.
깔끔한 선 두께의 변화와 기술적 주석,
단색(검은색 또는 파란색) 색상으로 도면다운 느낌을 살립니다.
{요소1}, {요소2}, {요소3}의 크기, 간격, 위치가 정확한 수치로 표시되고,
복잡한 관계도 간단하고 명확하게 표현됩니다.
전문 기술 도면 미학이며, 명확한 한글 주석과 높은 정확성이 특징입니다."
```

```
격자 배경의 청사진 도면으로 표현되며,
각 노드의 메모리 주소, 포인터, 값이 정밀하게 표시됩니다.
노드의 크기, 메모리 오프셋, 포인터 연결이 수치와 선으로 표현되고,
'0x7FFC...', 'left: 0x...', 'right: 0x...' 등 메모리 정보가 한글 설명과 함께 표시됩니다.
트리의 높이, 균형 상태, O(log n) 시간복잡도가 수학적 주석으로 추가되며,
컴퓨터 과학을 배우는 학생들이 내부 동작을 정확히 이해할 수 있는 설계입니다."
```

```
데이터베이스 설계자나 고급 학습자가 인덱스 최적화를 이해하는 데 도움이 됩니다.
정밀한 한글 라벨과 기술적 정확성이 최우선입니다."
```

#### 예제 4.3: 스택 메모리 레이아웃

```
메모리 낮은 주소에서 높은 주소로의 성장 방향이 화살표로 표시되고,

'Base Pointer (BP)', 'Stack Pointer (SP)' 위치가 명확하게 표시되며,
함수 호출 시의 푸시/팝 동작이 단계별로 표현됩니다.
메모리 주소는 16진수(0x...)로, 변수 크기는 바이트(4B, 8B) 단위로 표시되며,
시스템 프로그래밍이나 저수준 프로그래밍을 배우는 학생들을 위한 설계입니다."
```

## 🎨 Style 5: 인포그래픽 벡터

### 설명

매거진 수준의 전문 인포그래픽. 데이터와 정보를 시각적으로 매력적이고 이해하기 쉽게 표현합니다.

**최적 용도**:
- 통계 및 비교 데이터
- 성능 벤치마크
- 기능 비교표
- 프로젝트 성과 요약
- 학습 로드맵

**특징**:
- 통합된 차트와 그래프
- 아이콘 디자인
- 다채로운 색상 팔레트
- 전문적인 타이포그래피
- 인쇄 및 디지털 모두 적합

### 한국어 프롬프트 템플릿

```
"{주제}의 전문 인포그래픽 스타일 삽화입니다.
차트, 그래프, 데이터 시각화가 통합된 편집 품질의 벡터 일러스트레이션이며,
깔끔한 아이콘과 산세리프 타이포그래피를 사용합니다.
{색상1}, {색상2}, {색상3} 계열의 매거진 수준의 색상 팔레트를 적용하고,
{데이터1}, {데이터2}, {데이터3}이(가) 시각적으로 명확하게 비교 표현됩니다.
바 차트, 파이 차트, 라인 그래프 등 적절한 차트 형태가 선택되어 통합되며,
각 섹션의 제목과 설명이 명확한 한글로 표시됩니다.
인쇄와 디지털 출판 모두에 적합한 전문적이고 읽기 쉬운 설계입니다."
```

### 구체적 예제

```
'상승세', '안정적', '쇠락 추세' 등의 트렌드가 한글로 추가로 설명되며,
개발자 커뮤니티와 채용 시장 동향을 한눈에 파악할 수 있는 설계입니다.

```

## 🎨 Style 6: 포토리얼리스틱 3D 렌더

### 설명

사실적인 3D 렌더링으로 현실감 있게 표현하는 스타일입니다.

**최적 용도**:
- 하드웨어 및 물리적 장치
- 실제 사용 시나리오
- IoT 및 센서 설명
- 제품 레이아웃
- 현실 세계 적용 사례

**특징**:
- 사실적인 조명과 그림자
- 정확한 재질 표현
- 얕은 심도로 초점 표현
- 사진 품질의 렌더링
- 스튜디오 촬영 느낌

### 한국어 프롬프트 템플릿

```
"{주제}의 포토리얼리스틱 3D 렌더 이미지입니다.
사실적인 조명과 부드러운 그림자, 정확한 재질 질감과 반사가 표현되며,
초점을 위한 얕은 심도와 배경 흐림이 있습니다.
{요소1}, {요소2}, {요소3}이(가) 실제 물체처럼 정밀하게 렌더링되고,
스튜디오 사진 촬영 미학의 고해상도 제품 시각화입니다.
조명은 전문적인 스튜디오 조명으로, 각 요소의 세부사항과 텍스처가 명확합니다.
교육용으로 개발된 정확한 렌더링이며, 한글 라벨은 작고 명확하게 배치됩니다."
```

<a name="style-7"></a>
## 🎨 Style 7: 만화/교육용 코믹

### 설명

캐릭터와 스토리로 개념을 설명하는 만화 스타일. 친근하고 재미있습니다.

**최적 용도**:
- 단계별 튜토리얼
- 문제 해결 시나리오
- 실수와 해결방법
- 학습자 여정
- 초급 대상 개념 설명

**특징**:
- 캐릭터 일러스트레이션
- 말풍선과 생각 풍선
- 패널 레이아웃 (3-4개)
- 만화 선화
- 다이나믹한 표현

### 한국어 프롬프트 템플릿

```
"{주제}를 설명하는 교육용 만화 스타일 삽화입니다.
{캐릭터}이(가) {상황}을(를) 겪으며 단계적으로 개념을 설명하는 3-4 패널 만화 형식이며,
명확한 말풍선과 생각 풍선에 한글 대사와 설명이 포함됩니다.
만화풍의 깔끔한 선화로 {분위기} 느낌이 전달되고,
각 패널이 진행 상황을 명확하게 표현하는 구성입니다.
캐릭터의 표정, 몸짓, 반응이 생동감 있게 표현되며,
{학습포인트} 부분이 강조되는 시각적 강조(색상, 크기, 효과음)가 있습니다.
모든 연령의 학습자가 스토리를 통해 개념을 자연스럽게 이해할 수 있는 설계입니다."
```

### 구체적 예제

#### 예제 7.1: 디버깅 프로세스 (문제 해결)

```
"프로그래밍 디버깅 프로세스를 설명하는 교육용 만화입니다.
귀여운 개발자 캐릭터가 버그를 발견하고 찾아내고 수정하는 4개 패널 만화 형식이며,
각 패널에 명확한 한글 대사와 설명이 포함됩니다.

[패널 1] 캐릭터가 에러 메시지를 보고 놀라며 '어? 작동이 안 되는데?'라고 말함
[패널 2] 캐릭터가 코드를 자세히 들여다보며 '혹시... 이 변수?'라고 생각함
[패널 3] 캐릭터가 디버거를 사용해 실행을 추적하며 '아! 여기서 문제가!'라고 깨달음
[패널 4] 수정된 코드가 정상 작동하며 캐릭터가 즐거워하며 '해결했다!'고 외침

만화풍의 깔끔한 선화로 친근하고 긍정적인 느낌이 전달되고,
버그 수정의 성공에 대한 다양한 감정 표현(놀람 → 주의집중 → 깨달음 → 기쁨)이 시각화됩니다.
초급 프로그래머들이 디버깅을 체계적으로 접근하는 방법을 자연스럽게 학습하게 됩니다.
한글 대사와 시각적 표현으로 모든 학습자가 쉽게 이해할 수 있습니다."
```

```
[패널 2] 긴 줄의 사람들이 순서대로 기다리는 모습으로 답답한 표정
[패널 3] Task 코더가 복수의 테스크를 동시에 처리하는 '비동기 방식: 여러 일을 동시에!'
[패널 4] 여러 명이 각자 다른 일을 하는 효율적인 모습으로 만족스러운 표정

만화풍의 표현으로 추상적인 비동기 개념이 구체적인 생활 장면으로 표현되고,
각 패널의 효율성 차이가 시각적으로 명확하게 대비됩니다.
한글로 '기다림', '병렬 처리', '효율성' 등의 핵심 용어가 강조되며,
초급자도 비동기 프로그래밍의 필요성과 동작을 직관적으로 이해합니다."
```


## 🎨 Style 8: 그라디언트 현대 기술

### 설명

생동감 있는 그라디언트와 네온 글로우 효과로 미래지향적인 느낌을 전달합니다.

**최적 용도**:
- 최신 기술 주제 (AI, 블록체인, 양자 컴퓨팅)
- 차세대 기술 소개
- 혁신적인 개념 설명
- 고급 기술 주제
- 기술 컨퍼런스 슬라이드

**특징**:
- 생동감 있는 그라디언트 메시
- 네온 글로우 효과
- 어두운 배경
- 밝은 강조 색상
- 미래지향적 미학

### 한국어 프롬프트 템플릿

```
"{주제}의 현대적 그라디언트 기술 삽화입니다.
생동감 있는 그라디언트 메시와 색상 전환이 특징이며,
{색상1}에서 {색상2}로의 부드러운 그라디언트가 배경을 형성합니다.
은은한 네온 글로우 효과와 발광하는 선들이 미래 기술의 느낌을 전달하고,
어두운 배경에 밝은 액센트 색상이 강조됩니다.
{요소1}, {요소2}, {요소3}이(가) 네온 빛으로 강조되며 입체감 있게 표현됩니다.
현대 기술 산업 스타일의 전문적이고 트렌디한 디자인이며,
한글 라벨은 밝은 색상 또는 그라데이션으로 표시되어 전체 구성과 조화를 이룹니다."
```

### 구체적 예제

#### 예제 8.1: 머신러닝 신경망 시각화

```
"머신러닝 신경망 구조의 현대적 그라디언트 기술 삽화입니다.
입력층, 은닉층, 출력층이 파란색에서 보라색으로의 그라디언트로 표현되며,
각 노드가 네온 초록색과 파란색 글로우로 빛나며 강조됩니다.
노드 간의 가중치 연결이 발광하는 선으로 표현되고,
데이터가 흐르는 방향이 밝은 선으로 애니메이션되는 느낌으로 표시됩니다.
'입력', '계산', '예측' 등의 한글 라벨이 밝은 청색으로 표시되며,
신경망의 계층적 구조와 복잡한 계산 과정이 미래지향적으로 표현됩니다.
AI와 딥러닝의 최신 기술 트렌드를 강조하는 프레젠테이션 슬라이드에 적합합니다."
```

```
양자 컴퓨팅의 혁신성과 미래의 가능성을 시각적으로 전달하는 설계입니다.
고급 기술 세미나와 미래 기술 소개 강의에 매우 효과적입니다."
```
