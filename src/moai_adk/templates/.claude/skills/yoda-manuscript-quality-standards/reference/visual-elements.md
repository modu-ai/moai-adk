# 이미지 및 다이어그램

**목적**: 시각적 요소로 이해도 향상

---

## 1. 이미지 개수 제한

### 절(Section)당

- 최소: 1개
- 권장: 2-3개
- 최대: 4개

---

## 2. 핸드 드로잉 표준 템플릿

**필수 사용 템플릿** (`.claude/skills/yoda-educational-image-prompts/templates/standard-hand-drawn.md` 참조):

```
핸드 드로잉 삽화 스타일의 '[제목]' 인포그래픽입니다.
종이 질감 배경에 색연필로 그린 듯한 따뜻한 색감과 손으로 그린 선,
유기적인 화살표가 특징입니다.
[구체적 설명]

주요 키워드 및 요소:
- 스타일: 핸드 드로잉, 스케치 노트, 만화, 따뜻한 색연필 질감
- 배경: 종이 질감 (크림색 또는 베이지)
- 텍스트: 손글씨 스타일
- 요소: [구체적 요소]
- 전반적인 느낌: 친근하고 따뜻하며 정보 전달력이 높은 인포그래픽
```

---

## 3. Mermaid 다이어그램 활용

### 간단한 플로우는 Mermaid로

- 3-5단계 플로우차트
- 시퀀스 다이어그램
- 상태 다이어그램

### 이미지는 복잡한 개념만

- 6개 이상 컴포넌트
- Before-After 비교
- 터미널 화면 예시

---

## 4. 선택 기준

| 조건 | 권장 방법 |
|------|---------|
| **간단한 플로우 (3-5단계)** | Mermaid 다이어그램 |
| **복잡한 구조 (6개 이상)** | 핸드 드로잉 이미지 |
| **Before-After 비교** | 핸드 드로잉 이미지 |
| **터미널 화면** | 스크린샷 또는 이미지 |
| **개념 설명** | Mermaid 또는 이미지 |
