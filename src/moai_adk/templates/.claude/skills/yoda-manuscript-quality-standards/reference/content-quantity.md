**목적**: 일관되고 충실한 콘텐츠 제공

---

|---------|---------|---------|---------|
| **한글 단어** | 1,500 단어 | 2,000 단어 | 2,500 단어 |
| **문단** | 10 문단 | 15 문단 | 20 문단 |

---

## 3. 문단 구성 기준

### 각 문단

- 최소 3문장 이상
- 최대 7문장 이하
- 평균 80-120 단어
- 하나의 핵심 아이디어만 다룸

### 문장 길이

- 평균: 20-30 단어
- 최대: 50 단어 (복잡한 기술 설명)
- 단문과 장문 적절히 혼합

---

## 4. 깊이 있는 설명 기준

### 피상적 설명 (❌)

```
Claude Code는 AI 코딩 도구다. 자연어로 요청하면 코드를 생성한다.
```
(2문장, 20단어 - 너무 짧음)

### 깊이 있는 설명 (✅)

```
Claude Code는 Anthropic이 개발한 터미널 기반 AI 코딩 어시스턴트로, 2024년 중반에 정식 출시되었다 (1).
이 도구의 가장 큰 특징은 자연어로 요청을 받아 자율적으로 코드를 생성하고 실행하며,
오류가 발생하면 자동으로 디버깅까지 수행한다는 점이다 (1).
예를 들어, "Python으로 사용자 로그인 기능을 구현해줘"라고 요청하면
Claude Code는 필요한 모든 파일(예: login.py, test_login.py, requirements.txt)을 생성하고,
데이터베이스 연결 코드를 작성하며, 보안 검증 로직을 추가하고,
테스트 코드까지 자동으로 작성하여 실행한다 (1).

이러한 자동화는 전통적인 코드 자동완성 도구와 근본적으로 다르다.
GitHub Copilot과 같은 1세대 도구는 한 줄 또는 몇 줄의 코드만 제안하지만,
Claude Code는 전체 기능을 처음부터 끝까지 완성한다 (2).
실제로 Anthropic의 2025년 사례 연구에 따르면,
Claude Code를 사용한 개발자들은 평균적으로 2-3배의 생산성 향상을 경험했다고 보고했다 (3).
```
(5문단, 200+ 단어, 3개 인용 - 깊이 있고 구체적)
