**PART 1 (Ch 1-5)** 기초 학습용 챕터 작성 시 사용

**주요 특징**:

- 단계별 설명 (Step-by-step)

## 🏗️ 5가지 구성 요소

### 1. 문서 구조

```
총 구조:
├─ 도입 (1 섹션, 1-2 단락, 각 3-4 문장)
├─ 전환 (1-2 섹션, 각 1 단락, 각 2-3 문장)
├─ 본문 섹션 1 (2-3 단락, 각 4-5 문장)
├─ 본문 섹션 2 (2-3 단락, 각 4-5 문장)
├─ 본문 섹션 3 (2-3 단락, 각 4-5 문장)
└─ 요약 (1 섹션, 1 단락, 3-4 문장)

총 섹션: 6-7개
총 단락: 11-15개
평균 문장/단락: 3-4개
계층 깊이: H1 (Chapter) → H2 (Section) → H3 (Subsection)
```

### 2. 문체

**어조**: 친근하고 격려하는 톤
- "함께 배워봅시다"
- "천천히 따라해보세요"
- "잘하고 있어요!"

**종결어미**:
- "-요" (85%) - "설치해볼게요", "확인했어요"
- "-습니다" (15%) - "중요합니다", "필수입니다"

**능동태 비율**: 90% 이상
- ✅ "우리는 설치합니다" (능동태)
- ❌ "설치됩니다" (수동태)

**문장 길이**: 평균 18단어 (짧고 명확하게)

**전문 용어**: 한국어 우선 (영어 병기)
- "프롬프트 (Prompt)"
- "컨텍스트 (Context)"

### 3. 내용 전개

**배치 순서**:
1. 문제 제기 ("왜 필요한가?")
2. 솔루션 제시 ("이렇게 하면 됩니다")
3. 단계별 설명 (Step 1 → Step 2 → Step 3)
4. 확인 ("잘 작동하는지 확인해보세요")

**깊이 조절**:
- Level 1 (기초) 70%
- Level 2 (응용) 30%
- Level 3 (고급) 0%

**예제 복잡도**: 점진적 증가
- 기본 예제: 5줄
- 중급 예제: 10줄
- 고급 예제: 15줄

**연결성**: 이전 장 20% 참조
- "앞서 배운 X를 바탕으로..."

### 4. 조건

**글자 수**: 1500-2000자 (초급자 최적)

**코드 예제**: 3-5개
- 각 5-10줄
- 실행 가능한 완전한 코드
- 한국어 주석 포함

**대상 독자**: 초급 (0-1년)

**필수 요소**:
- 학습 목표 3개
- 체크포인트 3개 (각 단계 완료 시)
- 요약 1개
- 연습 문제 5개 (Level 1-5)

**선택 요소**:
- 트러블슈팅 1개
- 참고 자료 2-3개

### 5. 형식

**Markdown 구조**:
```markdown
# Chapter N: 제목 (H1)

## N-1: 섹션 제목 (H2)

### 학습 목표 🎯 (H3 + 이모지)

- 목표 1 (순서 없는 목록)
- 목표 2
- 목표 3

### 개념 설명

**핵심 개념**: ... (굵게 강조)

> 중요: ... (인용문으로 핵심 강조)

코드 예제:
```python
# 기본 예제 (언어 지정 필수, 한국어 주석)
def hello():
    print("Hello, World!")
```

✅ **체크포인트**: 이제 다음 단계로...

### 표 예시

| 항목 | 설명 | 예시 |
|------|------|------|
| A | 설명 A | 예시 A |

### 요약 📌

- 핵심 1
- 핵심 2
- 핵심 3

### 다음 장 미리보기 👀

다음 장에서는...
```

**아이콘 사용**:
- 🎯 학습 목표
- ✅ 체크포인트/완료
- ⚠️ 주의사항
- 💡 팁
- 📌 요약
- 👀 미리보기

**코드 블록**: 언어 지정 필수
- ```python
- ```javascript
- ```bash
- ```typescript

**강조**:
- **굵게**: 핵심 용어
- *기울임*: 부연 설명
- `코드`: 인라인 코드

## 📝 작성 프롬프트 템플릿

```
당신은 PART 1 (기초) 챕터를 작성하는 기술 저자입니다.

**어조**: 친근하고 격려하는 톤 ("-요" 85%, "-습니다" 15%)
**대상 독자**: 초급 (0-1년 경험)
**글자 수**: 1500-2000자

**필수 구조**:
1. 학습 목표 3개 🎯
2. 도입 (문제 제기 → 솔루션)
3. 본문 3개 섹션 (각 2-3 단락, 각 4-5 문장)
4. 코드 예제 3-5개 (각 5-10줄, 한국어 주석)
5. 체크포인트 3개 ✅
6. 연습 문제 5개 (Level 1-5) 📝
7. 요약 📌
8. 다음 장 미리보기 👀

**내용 전개**:
- 기초 70% + 응용 30%
- 예제 복잡도: 5줄 → 10줄 → 15줄
- 이전 장 20% 참조

**문체**:
- 능동태 90% ("우리는 설치합니다" > "설치됩니다")
- 문장 길이: 평균 18단어
- 용어: 한국어 (영어 병기) - "프롬프트 (Prompt)"

이제 다음 챕터를 작성하세요:
- **Chapter**: [챕터 번호와 제목]
- **주제**: [핵심 주제]
- **이전 장 요약**: [중복 제거용]
```
