# 장르 템플릿: introduction (도입형)

## 📋 용도

챕터 시작 부분, 독자 흥미 유발

**구조**: 문제제기 → 솔루션 제시 → 미리보기 → 다음 내용 연결

**특징**:
- 질문으로 시작
- 명확한 가치 제시
- 호기심 유발

---

## 🏗️ 5가지 구성 요소

### 1. 문서 구조

```
├─ 문제 제기 (1 단락, 3-4 문장)
├─ 솔루션 제시 (1 단락, 3-4 문장)
├─ 미리보기 (1 단락, 2-3 문장)
└─ 연결 (1 단락, 2 문장)

총 단락: 4개
총 문장: 10-13개
```

### 2. 문체

**어조**: 질문과 답변 형식

**종결어미**: 혼합 사용
- 질문: "~인가요?", "~일까요?"
- 답변: "-요", "-습니다"

**능동태**: 85% 이상

**문장 길이**: 평균 16단어

### 3. 내용 전개

**배치 순서**:
1. 문제 제기 ("이런 경험 있나요?")
2. 솔루션 제시 ("이렇게 하면 됩니다")
3. 미리보기 ("이 장에서 배울 내용")
4. 연결 ("지금 시작합니다")

### 4. 조건

**글자 수**: 300-500자

**대상 독자**: 모든 수준

**필수 요소**:
- 문제 제기 질문 1개
- 솔루션 명시
- 미리보기 2-3개 항목

### 5. 형식

```markdown
## N-1: 도입

### 이런 경험 있나요?

[문제 상황 설명]

### 해결 방법이 있습니다

[솔루션 제시]

### 이 장에서 배울 내용

- 항목 1
- 항목 2
- 항목 3

이제 시작해봅시다!
```
