# 장르 템플릿: hands-on-practice (실습형)

## 📋 용도

프로젝트, 직접 따라하기

**구조**: 준비 → Step 1 → Step 2 → Step 3 → 완성 확인

**특징**:
- 체크포인트
- 트러블슈팅
- 완성 기준 명확

---

## 🏗️ 5가지 구성 요소

### 1. 문서 구조

```
├─ 준비 (1 단락, 2-3 문장 + 목록)
├─ Step 1 (2 단락 + 코드, 각 3-4 문장)
├─ Step 2 (2 단락 + 코드, 각 3-4 문장)
├─ Step 3 (2 단락 + 코드, 각 3-4 문장)
└─ 완성 확인 (1 단락, 2-3 문장)

총 단락: 8개
코드 블록: 3개
체크포인트: 3개
```

### 2. 문체

**어조**: 실습 안내

**종결어미**: "-요", "-세요"

**능동태**: 95% 이상

**문장 길이**: 평균 17단어

### 3. 내용 전개

**배치 순서**:
1. 준비 (필요한 것)
2. Step 1 (기본 설정)
3. Step 2 (기능 구현)
4. Step 3 (완성)
5. 완성 확인

### 4. 조건

**글자 수**: 1000-1500자

**코드 예제**: 3개 (각 Step마다)

**필수 요소**:
- 체크포인트 3개
- 트러블슈팅 1-2개
- 완성 기준

### 5. 형식

```markdown
## N-1: 직접 만들어보기

### 준비 사항 📋
- 항목 1
- 항목 2

## Step 1: 기본 설정

```bash
# 명령어
```

✅ 체크포인트: ...

## Step 2: 기능 구현

```typescript
// 코드
```

⚠️ 트러블슈팅: ...

## Step 3: 완성

## 완성 확인 ✅

### 실행 결과
```
