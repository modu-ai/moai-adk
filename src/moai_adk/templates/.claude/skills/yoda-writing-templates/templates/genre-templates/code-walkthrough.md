# 장르 템플릿: code-walkthrough (코드 설명형)

## 📋 용도

기능 설명, 코드 기반 학습

**구조**: 목표 → Basic (5-10줄) → Intermediate (10-15줄) → Advanced (15-25줄) → 정리

**특징**:
- 점진적 공개
- 각 단계 실행 결과 필수
- 상세한 주석

---

## 🏗️ 5가지 구성 요소

### 1. 문서 구조

```
├─ 목표 (1 단락, 2-3 문장)
├─ Basic (1 단락 + 코드, 3-4 문장)
├─ Intermediate (1 단락 + 코드, 3-4 문장)
├─ Advanced (1 단락 + 코드, 4-5 문장)
└─ 정리 (1 단락, 2-3 문장)

총 단락: 5개
코드 블록: 3개
```

### 2. 문체

**어조**: 설명과 시연

**종결어미**: "-다", "-요"

**능동태**: 90% 이상

**문장 길이**: 평균 18단어

### 3. 내용 전개

**배치 순서**:
1. 목표 명시
2. Basic (기본 구조)
3. Intermediate (기능 추가)
4. Advanced (완전한 기능)
5. 정리 (핵심 요약)

### 4. 조건

**글자 수**: 800-1200자

**코드 예제**: 3개 (5-10줄, 10-15줄, 15-25줄)

**필수 요소**:
- 각 코드마다 실행 결과
- 상세한 주석
- 점진적 복잡도 증가

### 5. 형식

```markdown
## N-1: 코드로 배우기

### 목표

### Basic: 기본 구조

```python
# 기본 (5-10줄, 주석)
```

**실행 결과**:
```
Output...
```

### Intermediate: 기능 추가

```python
# 중급 (10-15줄)
```

### Advanced: 완전한 기능

```python
# 고급 (15-25줄)
```

### 정리
```
