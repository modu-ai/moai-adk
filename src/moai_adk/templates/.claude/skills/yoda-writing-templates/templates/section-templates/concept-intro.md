# 섹션 템플릿: concept-intro (개념 도입)

## 📋 용도

본문 시작, 핵심 개념 소개

**위치**: 학습 목표 직후 (H2 또는 H3 헤딩)

**길이**: 1-2단락 (각 50-100자)

---

## 🏗️ 5가지 구성 요소

### 1. 문서 구조

```
### [개념명]

**정의** (1 문장, 20-30자)

비유 또는 설명 (2-3 문장, 50-80자)

> 핵심: ... (인용문, 1 문장)

(총 1-2 단락)
```

### 2. 문체

**어조**: 명확하고 이해하기 쉽게

**종결어미**: "-다", "-습니다"

**능동태**: 85% 이상

**문장 길이**: 평균 20단어

### 3. 내용 전개

**배치 순서**:
1. 정의 (무엇인가?)
2. 비유 (이해하기 쉬운 예시)
3. 중요성 (왜 배워야 하는가?)
4. 전환 (다음 내용으로)

### 4. 조건

**글자 수**: 150-250자

**필수 요소**:
- 명확한 정의 1개
- 비유 또는 구체적 예시 1개
- 인용문으로 핵심 강조 (선택)

### 5. 형식

```markdown
### [개념명]

**정의**: [핵심 정의를 한 문장으로]

[비유 또는 구체적 설명 2-3 문장]

> **핵심**: [가장 중요한 점 1 문장]

[전환 문장: "이제 자세히 알아봅시다"]
```

---

## 📝 작성 프롬프트 템플릿

```
개념 도입 섹션을 작성하세요.

**구조**:
1. 정의 (1 문장, 20-30자)
2. 비유/설명 (2-3 문장)
3. 인용문 (핵심 1 문장)
4. 전환

**글자 수**: 150-250자

**조건**:
- 명확한 정의
- 이해하기 쉬운 비유
- 능동태 85% 이상

개념: [개념명]
대상 독자: [초급/중급/고급]
```
