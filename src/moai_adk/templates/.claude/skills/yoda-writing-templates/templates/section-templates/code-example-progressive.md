# 섹션 템플릿: code-example-progressive (코드 예제 - 3단계)

## 📋 용도

본문 중간, 기능 설명

**구조**: Basic → Intermediate → Advanced (점진적 복잡도 증가)

**특징**: 각 단계마다 완전한 실행 가능한 코드

---

## 🏗️ 5가지 구성 요소

### 1. 문서 구조

```
### 코드로 배우기

#### Basic: 기본 구조

코드 (5-10줄)
실행 결과

#### Intermediate: 기능 추가

코드 (10-15줄)
실행 결과

#### Advanced: 완전한 기능

코드 (15-25줄)
실행 결과

(3개 단계)
```

### 2. 문체

**어조**: 설명과 시연

**종결어미**: "-다", "-요"

**능동태**: 90% 이상

**문장 길이**: 평균 16단어

**주석**: 한국어, 명확하고 짧게

### 3. 내용 전개

**배치 순서**:
1. Basic (최소 기능)
2. Intermediate (기능 확장)
3. Advanced (완전한 구현)

**복잡도 증가**:
- Basic: 5-10줄
- Intermediate: 10-15줄
- Advanced: 15-25줄

### 4. 조건

**글자 수**: 코드 + 설명 = 600-1000자

**코드 줄 수**:
- PART 1: 5줄 → 10줄 → 15줄
- PART 2: 10줄 → 15줄 → 25줄
- PART 3: 10줄 → 20줄 → 30줄
- PART 4: 15줄 → 25줄 → 40줄

**필수 요소**:
- 각 단계마다 실행 결과
- 한국어 주석
- 완전한 코드 (복사해서 실행 가능)

### 5. 형식

```markdown
### 코드로 배우기

#### Basic: 기본 구조

```python
# 기본 예제 (5-10줄)
def basic_example():
    # 핵심 로직
    return result
```

**실행 결과**:
```
Output: ...
```

#### Intermediate: 기능 추가

```python
# 중급 예제 (10-15줄)
def intermediate_example():
    # 기능 확장
    return enhanced_result
```

**실행 결과**:
```
Output: ...
```

#### Advanced: 완전한 기능

```python
# 고급 예제 (15-25줄)
def advanced_example():
    # 완전한 구현
    return complete_result
```

**실행 결과**:
```
Output: ...
```
```
