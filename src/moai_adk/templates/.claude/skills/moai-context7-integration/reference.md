## Context7 MCP API Reference

### resolve-library-id

**Purpose**: Resolve user-friendly library name to Context7-compatible ID.

**Signature**:
```python
async def mcp__context7__resolve_library_id(libraryName: str) -> str:
    """
    Resolve library name to Context7 ID.
    
    Args:
        libraryName: User-friendly name (e.g., "fastapi", "react")
    
    Returns:
        Context7-compatible library ID (e.g., "/tiangolo/fastapi")
    
    Raises:
        LibraryNotFoundError: If library not found
    """
```

**Examples**:
```python
# Python libraries
await mcp__context7__resolve_library_id("fastapi")  # Returns: /tiangolo/fastapi
await mcp__context7__resolve_library_id("django")   # Returns: /django/django

# JavaScript libraries
await mcp__context7__resolve_library_id("react")    # Returns: /facebook/react
await mcp__context7__resolve_library_id("nextjs")   # Returns: /vercel/next.js

# Go libraries
await mcp__context7__resolve_library_id("gin")      # Returns: /gin-gonic/gin
await mcp__context7__resolve_library_id("gorm")     # Returns: /go-gorm/gorm
```

### get-library-docs

**Purpose**: Fetch documentation for specific library and topic.

**Signature**:
```python
async def mcp__context7__get_library_docs(
    context7CompatibleLibraryID: str,
    topic: str = "",
    page: int = 1
) -> str:
    """
    Fetch library documentation from Context7.
    
    Args:
        context7CompatibleLibraryID: Context7 ID (e.g., "/tiangolo/fastapi")
        topic: Focus topic (e.g., "routing dependency-injection")
        page: Page number for pagination (default: 1)
    
    Returns:
        Markdown-formatted documentation
    
    Raises:
        DocumentationNotFoundError: If documentation unavailable
    """
```

**Examples**:
```python
# Get FastAPI routing documentation
await mcp__context7__get_library_docs(
    "/tiangolo/fastapi",
    topic="routing dependency-injection",
    page=1
)

# Get React hooks documentation
await mcp__context7__get_library_docs(
    "/facebook/react",
    topic="hooks performance",
    page=1
)

# Get Gin middleware documentation
await mcp__context7__get_library_docs(
    "/gin-gonic/gin",
    topic="middleware authentication",
    page=1
)
```

### Cache Key Generation

```python
def generate_cache_key(
    library_id: str,
    topic: str,
    tokens: int,
    page: int
) -> str:
    """Generate unique cache key."""
    key_string = f"{library_id}:{topic}:{tokens}:{page}"
    return hashlib.sha256(key_string.encode()).hexdigest()[:16]
```

### Fallback Strategies

**Strategy 1: Alternative Library Names**
```python
def try_alternative_names(library_name: str) -> Optional[str]:
    """Try common alternative names."""
    alternatives = {
        "FastAPI": "fastapi",
        "fast-api": "fastapi",
        "Next.js": "nextjs",
        "next": "nextjs",
        "React.js": "react",
    }
    return alternatives.get(library_name)
```

**Strategy 2: Broader Topic Search**
```python
def generalize_topic(topic: str) -> str:
    """Generalize specific topic to broader category."""
    generalizations = {
        "dependency-injection": "architecture",
        "routing": "api",
        "hooks": "components",
        "middleware": "request-handling",
    }
    
    for specific, general in generalizations.items():
        if specific in topic:
            return general
    
    return ""  # No topic (fetch all)
```
