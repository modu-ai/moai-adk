# analytics-report
Generate blog analytics and performance report.

## Usage
```
/analytics-report [--period PERIOD] [--export FORMAT]
```

## Parameters
- `--period`: Time period (week, month, quarter, year)
- `--export`: Export format (json, pdf, csv)

## What It Does
1. Gathers analytics from all platforms
2. Calculates performance metrics
3. Identifies trending topics
4. Suggests optimization opportunities
5. Generates report

## Example
```bash
/analytics-report --period month --export pdf
```
