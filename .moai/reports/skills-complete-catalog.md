## Usage Patterns

### For Agents

Agents should reference skills using the explicit `Skill("skill-name")` pattern:

```markdown
## Required Skills

**Auto-loaded**:
- Skill("moai-domain-backend")
- Skill("moai-lang-python")

**Conditional** (load when needed):
- Skill("moai-essentials-debug") - For debugging sessions
- Skill("moai-essentials-perf") - For performance optimization
```

### For Skill Discovery

Skills can be discovered by:
1. **Category**: Browse by domain (Backend, Frontend, Security, etc.)
2. **Capability**: Search by key capability (API design, testing, etc.)
3. **Auto-trigger**: Keywords that activate skills automatically
4. **Recommendations**: Skills suggest related skills

3. Write SKILL.md with progressive disclosure

## Performance Considerations

### Skill Loading Strategy

**Auto-load** (eager):
- Core orchestration skills (moai-core-*)
- Foundation skills (moai-foundation-*)
- Primary domain skill for detected project type

**Conditional** (lazy):
- Language skills (load on file type detection)
- Specialty skills (load on explicit request)
- BaaS integrations (load on configuration detection)

**Token Optimization**:
- Skills use progressive disclosure (4 levels)
- Load Level 1 (Quick Reference) by default
- Load deeper levels on-demand
- Context7 integration provides just-in-time docs
