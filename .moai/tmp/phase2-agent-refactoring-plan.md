## Refactoring Pattern (from STEP_0_SETTING_REFACTOR_COMPLETE.md)

- Pseudo-code examples
- "Your task", "You will", "You should" phrasing

### Apply:
- Imperative verbs: "Read", "Write", "Check", "Ask", "Wait"
- Step-by-step structure
- IF/THEN conditions
- Error handling with timing
- Clear execution flow
