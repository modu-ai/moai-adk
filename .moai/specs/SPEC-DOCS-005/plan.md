## Phase 1: 기반 구축 (1-2주)

### 목표
- Next.js + Nextra 기술 스택 완전 설정
- 기본 문서 구조와 네비게이션 구현
- 핵심 페이지 프로토타입 개발

### 세부 계획

**1.1 기술 스택 설정**
- Next.js 15.1.8 설치 및 설정
- Nextra 3.3.1 테마 커스터마이징
- TypeScript 설정 및 타입 정의
- ESLint + Prettier 코드 품질 설정

**1.2 기본 구조 구현**
- 디렉토리 구조 표준화
- 메타데이터 및 SEO 설정
- 다크/라이트 모드 지원
- 검색 기능 설정

**1.3 핵심 페이지 프로토타입**
- 메인 페이지 뼈대
- 시작하기 섹션 기본 구조
- 네비게이션 및 메뉴 시스템
- 푸터 및 헤더 컴포넌트
