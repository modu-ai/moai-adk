- **EC-01**: 사용자가 `.moai/project/db/` 내 파일을 수동 편집한 후 `/moai db init` 재실행 → AC-009 의 idempotent 보호 동작 (SPEC-DB-CMD-001 Phase 3 에 위임, skeleton 수준).
